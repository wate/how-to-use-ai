Claude まとめ
================

以下の内容は2026年現在AnthropicのClaudeサービスに関する最新の情報をまとめたものです。

サービス概要
---------------------------

Anthropic が提供する大規模言語モデルを使った対話型AIサービス。  
高い安全性と誠実な応答を重視した設計が特徴で、長文の読解・文章作成・コーディング・分析など幅広いタスクに対応する。

プランと特徴
---------------------------

| プラン      | 月額(目安)  | 主な特徴                                                                                              |
| ----------- | ----------- | ----------------------------------------------------------------------------------------------------- |
| 無料        | 0円         | Claude Sonnet 4.6 を利用回数制限付きで使用可。プロジェクト作成は5つまで                              |
| Pro         | 約3,000円   | Claude Sonnet 4.6 / Opus 4.6 を優先的に利用可。利用回数が大幅に増加。プロジェクト数制限なし          |
| Team        | 約4,500円   | Pro の機能に加え、チームメンバーとの共有・管理機能が追加。管理コンソールで組織全体を一元管理          |
| Enterprise  | 別途        | カスタムコンテキスト・SSO・高度なセキュリティ・SLA など企業向けの機能を提供                          |

主要機能
---------------------------

### モデルラインナップ

| モデル              | 用途                                    | 対象プラン                        |
| ------------------- | --------------------------------------- | --------------------------------- |
| Claude Sonnet 4.6   | 日常タスクの高速デフォルト              | 全ユーザー(無料含む・回数制限あり)|
| Claude Opus 4.6     | 高度な推論・複雑な分析・長文処理        | Pro 以上                          |

- **長文コンテキスト**: 最大20万トークンのコンテキストウィンドウ。長い文書・コードベースを一度に処理可能
- **Projects(プロジェクト機能)**: カスタム指示・情報源ファイル・チャット履歴をまとめて管理できる機能。無料プランは5プロジェクトまで
- **画像入力**: 画像をアップロードして内容を分析・説明させることが可能。`⚠️ 画像生成には非対応`
- **ファイル添付**: PDF・Word・テキストなどのファイルをアップロードして内容を参照させることが可能
- **Web 検索**: リアルタイムの情報をWeb上から取得して回答に反映させることが可能。`⚠️ 利用可否はプランにより異なる`
- **Artifacts**: コード・文書・SVGなどをチャット画面の右側にプレビュー表示しながら作成・編集できる機能

Projects(プロジェクト機能)
---------------------------

### 概要

カスタム指示・情報源ファイル・チャット履歴を一箇所にまとめて管理できる機能。  
プロジェクトを選択してチャットを始めると、登録したカスタム指示と情報源ファイルが自動的に参照される。

### 利用と作成の条件

| 操作                           | 必要なプラン                                              |
| ------------------------------ | --------------------------------------------------------- |
| プロジェクトを**作成**する     | 無料プランでも可。ただし作成数は **5つまで**              |
| プロジェクトを**共有**する     | Team 以上                                                 |
| 情報源ファイルを登録する       | 無料プランでも可。ファイル数の上限なし(コンテキスト上限内)|

### 主な設定項目

- **カスタム指示**: プロジェクト全体に適用されるルール・ペルソナ・制約を設定する
- **情報源ファイル**: PDF・Word・テキストなどのファイルをアップロードし、回答の根拠として参照させる
- **チャット履歴**: プロジェクト内のチャットは一覧で管理でき、過去のやり取りを参照しやすい

### 現時点の制限

- 無料プランはプロジェクト作成数が **5つまで**
- 情報源ファイルの総量はコンテキストウィンドウ(20万トークン)に収まる範囲
- プロジェクトの共有は Team 以上のプラン
- 画像生成機能は非対応(画像の読み取り・分析は可能)

データコントロール・プライバシー
---------------------------

### 基本的な考え方

Claude はデフォルトでユーザーのチャットデータをサービスの改善やAIモデルの品質向上に利用する仕組みになっている。  
ただし、ユーザーは設定からデータ利用をオプトアウトすることができる。

### プランごとのデフォルト設定

| プラン                    | デフォルト                           | 変更可否             |
| ------------------------- | ------------------------------------ | -------------------- |
| 個人(無料 / Pro)          | 学習に**使用される**(オプトアウト可) | ユーザーが手動でオフ |
| Team / Enterprise         | 学習に**使用されない**               | 管理者が一括管理     |

### オプトアウトの方法

#### ① 設定からデータ利用をオフにする

「設定」→「プライバシー」→「会話の内容をモデルのトレーニングに使用する」をオフに切り替える。

#### ② インコグニートモードを使う

インコグニートモードでのチャットは保存されず、モデルの学習にも使用されない。  
ただし、チャット履歴も残らないため、過去のやり取りを参照したい場合は通常モードを使用する。

### 収集されるデータの種類

- Claude とのチャット内容(テキスト・画像・ファイル)
- AIが生成したコンテンツ
- 利用環境の情報(デバイス・ブラウザ・OS)

### 注意点

- オプトアウト設定と履歴削除は別の概念であり、オプトアウトだけでは過去データは残る
- 機密情報・個人情報の入力を避ける運用上の対策も重要
- Team / Enterprise プランでは組織のデータが学習に使用されないことがデフォルトで保証される
