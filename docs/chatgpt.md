ChatGPT まとめ
================

以下の内容は2026年現在OpenAIのChatGPTサービスに関する最新の情報をまとめたものです。

サービス概要
---------------------------

OpenAIが提供する大規模言語モデルを使った対話型AIサービス。  
2022年11月に公開されて以来急成長し、2024年末時点で週間アクティブユーザー数は3億人を超えている。

プランと特徴
---------------------------

| プラン           | 対象   | 主な特徴                                                     |
| ---------------- | ------ | ------------------------------------------------------------ |
| Free             | 個人   | GPT-5.3 Instant(デフォルト)利用可。プロジェクト機能利用可    |
| Plus / Go        | 個人   | GPT-5.3 Instant + GPT-5.5 Thinking、Canvas、Deep Research    |
| Pro              | 個人   | GPT-5.5 Pro を含む最上位モデルへのフルアクセス               |
| Team             | チーム | 共有プロジェクト、コネクター、高度な管理機能                 |
| Enterprise / Edu | 法人   | RBAC、コンプライアンス対応(ISO 27001 等)、エージェントモード |

主要機能
---------------------------

### モデルラインナップ

| モデル           | 用途                   | 対象プラン       |
| ---------------- | ---------------------- | ---------------- |
| GPT-5.3 Instant  | 日常タスクのデフォルト | 全ユーザー       |
| GPT-5.5 Thinking | 高度な推論・複雑タスク | Plus 以上        |
| GPT-5.5 Pro      | 最高難度の長期タスク   | Pro / Enterprise |

- **GPT-5.3 Instant**：全ユーザーのデフォルトモデル。ハルシネーションを前世代比で最大26.8%削減。過度に慎重・説教的なトーンを改善し、より自然な会話スタイルを実現(2026年3月〜)
- **GPT-5.5 Thinking / Pro**：コーディング・データ分析・マルチステップ研究に強い最上位モデル(2026年4月〜、API 提供も開始)
- **GPT-5.3 Codex**：エージェント型コーディング特化モデル。前世代比25%高速化。SWE-Bench Pro で最高水準を達成

### その他の主要機能

- **Canvas**：ドキュメント・コード・レイアウトのリッチ編集インターフェイス。`⚠️ 無料プランでは利用制限あり`
- **ChatGPT エージェント**：ウェブ操作・フォーム入力・スプレッドシート編集などのマルチステップタスクを自動実行。`⚠️ Plus 以上`
- **Deep Research**：複数ソースを統合した深掘りリサーチ(レガシーモードは2026年3月廃止)。`⚠️ Plus 以上`
- **コネクター**：Gmail・Google カレンダー・Microsoft Teams・GitHub・Dropbox など60以上のアプリと連携。`⚠️ Plus 以上`
- **音声モード(Advanced Voice Mode)**：リアルタイム音声会話。ウェブ・iOS・Android・Windows で利用可。`⚠️ 無料プランでは回数制限あり`
- **Codex**：コーディングエージェント。macOS アプリから複数タスクを並列管理可。`⚠️ Plus 以上(無料・Go は限定的な試用のみ)`
- **Memory**：会話を跨いでユーザーの好みや文脈を記憶(オプトイン)。`⚠️ 無料プランでは上限あり`

## 4. データコントロール・プライバシー

---------------------------

### 基本的な考え方

ChatGPT・Codex・Sora などの個人向けサービスでは、ユーザーのコンテンツがモデルのトレーニングに使用されることがある。ただし、ユーザーはオプトアウトを選択することで、新しいやり取りをモデルの学習に使用されないよう設定できる。

### プランごとのデフォルト設定

| プラン                  | デフォルト                           | 変更可否               |
| ----------------------- | ------------------------------------ | ---------------------- |
| Free / Plus / Pro       | 学習に**使用される**(オプトアウト可) | ユーザーが手動でオフ   |
| Team / Enterprise / API | 学習に**使用されない**               | 管理者がオプトインも可 |

> 法人向け製品(ChatGPT Business・Enterprise・API)の入出力は、デフォルトではモデルのトレーニングに使用されない。

### オプトアウトの方法

2026年時点では、オプトアウトには「設定画面から行う方法」と「プライバシーポータルから申請する方法」の2種類がある。今後の会話だけ学習から外したい場合は設定画面、過去のデータも含めて除外したい場合はプライバシーポータル、特定の会話だけ学習対象外にしたい場合は一時チャットが適している。

#### ① 設定画面から行う方法(即時反映)

プロフィールアイコン →「設定」→「データコントロール」→「すべての人のためにモデルを改善する」をオフにする。

#### ② プライバシーポータルから申請する方法(過去データも対象)

OpenAIプライバシーポータル(https://privacy.openai.com/policies)にアクセスし「Make a Privacy Request」をクリック。「私のコンテンツでトレーニングしないでください」を選択してメールアドレスを入力し、確認メールからログインしてリクエストを送信する。  
反映完了までに数日〜数週間かかるため、即時性が求められる場合は設定画面からの変更が現実的。

#### ③ 一時チャット

一時チャットの会話は履歴に表示されず、メモリを利用・作成せず、モデルのトレーニングにも使用されない。特定の会話だけ学習対象から外したい場合に有効。

### 注意点

- オプトアウトは今後のコンテンツに対して適用されるものであり、すでに学習に用いられたデータがモデルから削除されるわけではない。
- 設定画面からオプトアウトを有効にすると、チャット履歴の保存もオフになる場合がある。ただし現在は履歴を保持したままオプトアウトのみを設定することも可能。
- オプトアウト後も機密情報の入力にはリスクが伴うため、設定だけでなく入力内容の管理も重要。

GPTs(カスタム GPT)
---------------------------

### 概要

ユーザーが独自の指示・ペルソナ・ナレッジファイルを設定した、再利用可能なカスタム AI アシスタントを作成できる機能。  
2023年11月に登場。プログラミング不要でノーコードで作成でき、作成した GPT は GPT Store で公開・共有も可能。

### 利用と作成の条件

| 操作                                        | 必要なプラン                   |
| ------------------------------------------- | ------------------------------ |
| 他のユーザーが公開した GPT を**利用**する   | 無料プランでも可(回数制限あり) |
| 独自の GPT を**作成**する                   | Go 以上 `⚠️ 無料プランは不可`   |
| GPT Store での**収益化**に参加する          | Plus 以上                      |
| ワークスペース内限定の GPT を管理・展開する | Business / Enterprise          |

### 主な設定項目

- **指示(Instructions)**：GPT が毎回参照するルール・ペルソナ・制約を設定する
- **ナレッジファイル**：PDF・ドキュメント・画像などをアップロードし、回答の根拠として参照させる(最大20ファイル)
- **Actions**：外部 API と連携し、天気取得・データベース参照・フォーム送信などの処理を GPT 内から実行できる `⚠️ Plus 以上`
- **公開範囲**：「自分だけ」「リンクを持つ人のみ」「GPT Store に公開」の3段階から選択可能

### GPT Store

他のユーザーが作成・公開した GPT を検索・利用できるプラットフォーム。  
文書作成・コーディング・画像生成・語学学習など多様なカテゴリの GPT が公開されている。

### GPTs vs プロジェクト機能の使い分け

| 観点             | GPTs                                 | プロジェクト                       |
| ---------------- | ------------------------------------ | ---------------------------------- |
| 主な目的         | 再利用可能なカスタム AI の作成・配布 | チャット・ファイル・指示の一元管理 |
| 共有             | GPT Store または URL で広く共有可    | チームメンバーとの共同編集         |
| 外部連携         | Actions で外部 API に接続可          | コネクター(Gmail 等)と連携         |
| 向いているケース | 定型タスクの自動化、社内ツールの配布 | 長期プロジェクトの文脈管理         |

プロジェクト機能
---------------------------

### 概要

関連するチャット・ファイル・カスタム指示を一箇所にまとめて管理する「ワークスペース」機能。  
2024年12月に有料ユーザー向けに開始され、2025年9月4日から全ユーザー(無料含む)に開放された。

### 基本的な仕組み

- プロジェクト内のすべてのチャットが、同じファイルとカスタム指示を共有する
- 「どのモデルを使うか」「どのトーンで返答するか」などをプロジェクト単位で設定可能
- プロジェクト専用のメモリ(Memory)がオンになり、プロジェクト内の過去チャットを参照して応答を最適化する

### ファイルのアップロード上限

| プラン            | 上限                      |
| ----------------- | ------------------------- |
| Free              | 5ファイル / プロジェクト  |
| Go / Plus / Edu   | 25ファイル / プロジェクト |
| Team(Business)    | 40ファイル / プロジェクト |
| ※同時アップロード | 最大10ファイル            |

### 共有プロジェクト(Shared Projects)

2025年10月から全プラン(無料含む)に提供開始。チームで同一プロジェクトを共同編集可能。

- ファイルと指示をメンバーが追加でき、ChatGPT が共有コンテキストを参照して回答する
- ビジネスプランでは RBAC(ロールベースアクセス制御)により共有範囲を管理可能 `⚠️ RBAC は Business / Enterprise のみ`
- 削除されたプロジェクトのデータは30日以内に削除(法的保持義務を除く)

### 主なユースケース

- **コーディング・コードレビュー**：コードベースを理解させた上で質問やドキュメント生成
- **長期プロジェクト管理**：月次レポート・提案書の一貫したトーンと文脈を維持
- **学習**：授業資料を添付して個別問題演習や要約を繰り返し活用
- **クライアントワーク**：契約書・提案資料・メモを一元管理
