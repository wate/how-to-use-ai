Gemini まとめ
================

以下の内容は2026年現在GoogleのGeminiサービスに関する最新の情報をまとめたものです。

サービス概要
---------------------------

Google が提供する大規模言語モデルを使った対話型AIサービス。  
Gemini モデルファミリー(Flash / Pro 等)を基盤とし、Google Workspace・Google 検索・Google フォトなど Google サービス全体への深い統合が特徴。

プランと特徴
---------------------------

| プラン                 | 月額(目安) | 主な特徴                                                                                                                |
| ---------------------- | ---------- | ----------------------------------------------------------------------------------------------------------------------- |
| 無料                   | 0円        | Gemini 3 Flash(デフォルト)。Gemini 3.1 Pro も回数制限付きで試験的に利用可。15GB ストレージ                              |
| Google AI Plus         | 約1,200円  | Gemini 3.1 Pro へのアクセス拡大。思考モード最大90回/日・Proモード最大30回/日。200GB ストレージ                          |
| Google AI Pro          | 約2,900円  | Gemini 3.1 Pro・Deep Research・100万トークンコンテキスト。思考モード最大300回/日・Proモード最大100回/日。2TB ストレージ |
| Google AI Ultra        | 約36,400円 | Deep Think・Veo 3.1 動画生成・Gemini Agent(米国のみ)など最上位機能へのフルアクセス。思考モード最大1,500回/日            |
| Google Workspace(法人) | 別途       | 2025年1月から Gemini 機能が標準搭載。管理コンソールで組織全体のアクセス権限とデータポリシーを一元管理                   |

> ※ 2025年に「Google One AI プレミアム」から「Google AI Pro」に改称。同時に下位プランとして「Google AI Plus」が新設された。

主要機能
---------------------------

### モデルラインナップ

| モデル                    | 用途                                | 対象プラン                                       |
| ------------------------- | ----------------------------------- | ------------------------------------------------ |
| Gemini 3 Flash            | 日常タスクの高速デフォルト          | 全ユーザー(無料含む)                             |
| Gemini 3.1 Pro            | 高度な推論・コーディング・長文処理  | 無料は回数制限付き試験提供。安定利用は Plus 以上 |
| Gemini 3.1 Pro Deep Think | 最高難度の推論タスク(192k トークン) | Ultra のみ `⚠️ 現時点は米国・英語のみ`            |

- **コンテキスト窓 100万トークン**：大規模文書・コードベース全体を一度に処理可能。`⚠️ Pro 以上`
- **Deep Research**：複数ソースを横断した深掘りリサーチ。無料プランでは月5回まで。`⚠️ 安定利用は Plus 以上`
- **Google Workspace 統合**：Docs・Sheets・Gmail・Drive のサイドパネルから直接 Gemini を呼び出せる。`⚠️ 一部機能は Plus 以上 / Workspace プラン`
- **Gemini Live**：リアルタイム音声会話。`⚠️ 無料プランでは回数制限あり。Gems との連携は現時点で未対応`
- **Canvas**：ドキュメント・コードのリッチ編集インターフェイス。無料プランでも基本利用可。`⚠️ 高度な機能は制限あり`
- **画像生成(Nano Banana Pro)**：テキストから画像を生成。イラスト・写真風画像などに対応。無料プランでは1日100枚まで。`⚠️ 安定的な高品質生成は Plus 以上`
- **動画生成(Veo 3.1)**：テキストや写真から動画を生成。`⚠️ Ultra のみ(Plus / Pro は Veo 3.1 Lite の限定トライアル)`
- **Opal**(2025年12月〜)：ノーコードで AI ミニアプリを Gem として構築できるインターフェイス。`⚠️ 有料プラン`

Gems(カスタム AI エキスパート)
---------------------------

### 概要

ユーザーが独自の指示・ペルソナ・ナレッジファイルを設定した、再利用可能なカスタム AI アシスタントを作成できる機能。  
2024年8月にローンチされ、Gemini Advanced(現 Google AI Pro)の最も利用されている機能の一つ。

### 利用と作成の条件

| 操作                         | 必要なプラン                                                                                                                                |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| 既製 Gem を**利用**する      | 無料プランでも可(回数制限あり)                                                                                                              |
| 独自の Gem を**作成**する    | 無料プランでも可(2025年3月〜)。ただし作成数・利用回数・使用モデルに制限あり。`⚠️ 上限数は公式非公開で変動する可能性あり(目安：最大10個程度)` |
| Gem を組織内外に**共有**する | Google AI Plus 以上(2025年9月〜)`⚠️ 無料プランは不可`                                                                                        |
| ビジネス向け既製 Gem を利用  | Google Workspace アドオン                                                                                                                   |

### 主な設定項目

- **指示(Instructions)**：Gem が毎回参照するルール・ペルソナ・制約を設定する
- **ナレッジファイル**：最大10ファイルをアップロードし、回答の根拠として参照させる
- **Google Drive 連携**：Drive 上のドキュメントを参照先に指定でき、ファイル更新が即座に反映される(Living Document)`⚠️ Plus 以上`
- **Magic Wand**：下書き指示を AI が自動でブラッシュアップする補助機能
- **プレビュー**：Gem 作成画面でリアルタイムに動作確認できる

### ビジネス向け既製 Gem(Google Workspace)

| Gem 名              | 用途                                   |
| ------------------- | -------------------------------------- |
| Marketing insights  | マーケティング分析・顧客獲得コスト算出 |
| Sales pitch ideator | 営業資料作成                           |
| Hiring consultant   | 採用支援・面接質問生成                 |
| Outreach specialist | 顧客向けパーソナルメッセージ作成       |
| Copy creator        | ブランドに沿ったコンテンツライティング |
| Sentiment analyzer  | フィードバックの感情分析               |

### 現時点の制限

- Gemini Live(音声モード)との連携は未対応
- 画像生成は Gem から直接利用不可
- 外部 API 連携(ChatGPT の Actions に相当する機能)は未提供
- Gem の作成・編集はウェブアプリのみ(モバイルは使用のみ)

Notebooks(NotebookLM 統合)
---------------------------

### 概要

2026年4月8日、Gemini アプリに NotebookLM の機能を直接取り込んだ新セクション「**Notebooks(ノートブック)**」が正式導入された。  
Gemini のサイドバーからノートブックを作成・管理でき、Gemini と NotebookLM が単一の知識ベースを双方向リアルタイムで同期する。

### 統合の経緯

| 時期          | 内容                                                                                      |
| ------------- | ----------------------------------------------------------------------------------------- |
| 2025年12月    | NotebookLM のノートブックを Gemini チャットのソースとして添付できる機能が一般公開         |
| 2026年1月     | Google Workspace への段階展開開始                                                         |
| 2026年4月8日  | Gemini サイドバーへの Notebooks セクション統合を正式発表(Ultra / Pro / Plus から順次提供) |
| 2026年4月18日 | 無料ユーザーへの提供開始                                                                  |

### 主な機能

- **チャットからの直接参照**：Gemini チャットの「＋」メニューから NotebookLM のノートブックをソースとして選択し、その内容を参照しながら回答させることができる(2025年12月〜)
- **統合ナレッジベース**：Gemini 側でソース(PDF・URL・Google Docs 等)を追加すると NotebookLM 側にも即座に反映され、逆も同様
- **グラウンディング回答**：指定されたソースのみを根拠に回答するため、ハルシネーションを抑制した高精度な要約・Q&A が可能
- **カスタム指示**：ノートブック単位で Gemini の応答トーンや方針を設定できる
- **Canvas への引き継ぎ**：ノートブックの内容をそのまま Canvas へ渡して長文執筆に活用可能
- **音声概説・動画インフォグラフィック**：NotebookLM 側で同じ資料から音声解説や動画を生成できる
- **チャット履歴の追加**：Gemini との過去の会話を「ノートブックに追加」でナレッジベース化できる

### ノートブックの上限

| プラン    | ノートブック数 | 1ノートブックあたりのソース数     |
| --------- | -------------- | --------------------------------- |
| 無料      | 100            | 50ソース(1ソースあたり50万語まで) |
| Plus 以上 | 500            | 300〜600ソース                    |

### 利用上の注意

- 現時点では**個人の Google アカウントのみ**対応(仕事用・学校用アカウントは未対応)`⚠️`
- Gemini との会話が自動的に NotebookLM のソースに混入する場合があるため、「リサーチ専用」と「作業用」でノートブックを分けて運用することを推奨
- NotebookLM 側のポリシーが適用されるため、HIPAA 等の規制が厳しい業界での利用は事前に法務確認が必要

### Gemini vs NotebookLM の役割分担

| 観点             | Gemini                           | NotebookLM                                       |
| ---------------- | -------------------------------- | ------------------------------------------------ |
| 情報源           | Web 全体 ＋ 指定ソース           | 指定ソースのみ(グラウンディング)                 |
| 強み             | 発想・文章生成・最新情報との統合 | 高精度な資料分析・引用付き回答                   |
| 出力形式         | テキスト・Canvas・画像・動画     | 音声概説・動画・クイズ・インフォグラフィック     |
| 向いているケース | 戦略立案・外部情報との統合       | 社内資料の分析・ハルシネーション抑制が必要な場面 |

データコントロール・プライバシー
---------------------------

### 基本的な考え方

Geminiはデフォルトでユーザーのチャットデータをサービスの改善やAIモデルの品質向上に利用する仕組みになっている。
ただし、ユーザーは「Gemini アプリ アクティビティ」の設定を変更することで、チャット内容がAIモデルの学習に利用されない状態にできる。

### プランごとのデフォルト設定

| プラン                                 | デフォルト                           | 変更可否             |
| -------------------------------------- | ------------------------------------ | -------------------- |
| 個人(無料 / AI Plus / Pro / Ultra)     | 学習に**使用される**(オプトアウト可) | ユーザーが手動でオフ |
| Google Workspace(法人対象エディション) | 学習に**使用されない**               | 管理者が一括管理     |

> Workspace の対象エディション(Business / Enterprise 等)では、企業向けデータ保護(enterprise-grade data protections)がデフォルトで適用される。

### オプトアウトの方法

#### ① Gemini アプリ アクティビティをオフにする(即時反映)

「設定とヘルプ」→「データとプライバシー」→「Gemini アプリ アクティビティ」をオフに切り替える。モバイルアプリでは「メニュー(三本線)」→「設定」→「データとプライバシー」から同様に設定可能。  
反映後、通常48〜72時間でGoogleのシステム全体に変更が反映される。

#### ② Googleアカウントの「マイアクティビティ」から削除(過去データも対象)

[マイアクティビティ](https://myactivity.google.com)からGeminiのアクティビティ履歴を個別または一括で削除できる。  
なお、オプトアウト設定は「今後のデータ利用を拒否する」ものであり、過去データを完全に消去するには別途削除申請が必要。

#### ③ チャット履歴の自動削除設定

「Gemini アプリ アクティビティ」の自動削除設定から、一定期間が経過したデータを自動的に削除できる。選択肢は 3ヶ月 / 18ヶ月 / 36ヶ月 / 自動削除なし で、デフォルトは18ヶ月。

### 収集されるデータの種類

- Gemini とのチャット内容(テキスト・画像・音声)
- Gemini Live の音声入力・会話の文字起こし
- AIが生成したコンテンツ(画像・動画・要約など)
- 利用環境の情報(デバイス・ブラウザ・OS・位置情報)
- Google Maps・Photos・YouTube などの連携サービスの情報

### 注意点

- 収集データの一部は品質評価のため、人間のレビュアーが匿名化した上で確認する場合がある。レビュアーが確認したデータはアカウント履歴とは別に管理され、削除しても一定期間保存される。
- GmailやGoogle Driveなどの個人データがモデル学習に使用されるかどうかは公式には明確にされていない。
- オプトアウト設定と履歴削除は別の概念であり、オプトアウトだけでは過去データは残る。
- 設定だけでなく、機密情報・個人情報の入力を避ける運用上の対策も重要。
