AI活用事例
===========================

日常で使えるAIの活用方法をまとめています。  
難しい使い方ではなく、普段の生活でそのまま使える内容に絞っています。

[活用事例](examples/README.md)
---------------------------

### 1. [文章作成](examples/writing/README.md)

- メール返信文の作成
- ブログ記事・SNS投稿
- PR文・説明文の作成
- 下書き・たたき台生成

### 2. [画像生成・加工](examples/image-work/README.md)

- キーワード・テーマからイラストを生成する
- 既存の写真を加工・編集する
- テキスト・データから図解・ダイアグラムを生成する

### 3. [お金の管理](examples/money-management/README.md)

- 月次の支出振り返り
- 無駄の可視化
- 改善アクション提案

### 4. [記録・ふりかえり](examples/daily-logs/README.md)

- 日記・行動ログ
- 週次／月次の振り返り
- 自己改善の気づき

### 5. [相談・考え整理](examples/decision-support/README.md)

- 何に迷っているかの整理
- 選択肢・判断軸の提示
- タスクの優先順位付け
- 内容の確認やリスクの洗い出し
- 次のアクション明確化

### 6. [見直し・整理](examples/review/README.md)

- 月次・四半期の振り返り
- 大掃除・整理
- 各種見直し(契約・固定費など)
