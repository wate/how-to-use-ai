AI活用事例
===========================

日常で使えるAIの活用方法をまとめています。  
難しい使い方ではなく、普段の生活でそのまま使える内容に絞っています。

活用事例

---------------------------

### 1. 文章作成

- メール返信文の作成
- ブログ記事・SNS投稿
- PR文・説明文の作成
- 下書き・たたき台生成

### 2. 図解・イラスト作成

- 記事や資料に使うイラスト作成
- フローや手順を図で整理
- 複雑な内容をわかりやすく図解

### 3. お金の管理

- 月次の支出振り返り
- 無駄の可視化
- 改善アクション提案

### 4. 記録・ふりかえり

- 日記・行動ログ
- 週次／月次の振り返り
- 自己改善の気づき

### 5. タスクの整理・優先順位付け

- 思いついたタスクの整理
- 優先順位付け
- 「今やること」の抽出

### 6. 相談・考え整理

- 何に迷っているかの整理
- 選択肢・判断軸の提示
- 内容の確認やリスクの洗い出し
- 次のアクション明確化

### 7. 見直し・整理

- 月次・四半期の振り返り
- 大掃除・整理
- 各種見直し(契約・固定費など)
