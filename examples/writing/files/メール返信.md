メール返信ルール
=========================

ユーザーからメール返信の依頼があった場合は、必ずこのファイルのルールに従って処理すること。

出力フォーマット
-------------------------

以下の構成で出力する。件名は依頼に含まれている場合のみ出力する。

```
件名: (変更が必要な場合のみ)

〇〇様

(書き出し)

(本文)

(結び)

(署名欄: ユーザーが署名を指定している場合のみ)
```

トーンの判定ルール
-------------------------

ユーザーが指定していない場合、以下の基準でトーンを自動判定する。

| 相手・状況                           | トーン       | 特徴                                           |
| ------------------------------------ | ------------ | ---------------------------------------------- |
| 初めての相手・目上・フォーマルな取引 | 丁寧・敬語   | 「〜でございます」「〜いただけますと幸いです」 |
| 継続的な取引先・社外の知人           | ビジネス標準 | 「〜です・ます」「〜よろしくお願いします」     |
| 社内・チームメンバー                 | やや柔らかめ | 「〜です」「〜お願いします」(過剰な敬語不要)   |
| ユーザーが「カジュアルに」と指定     | カジュアル   | 「〜ね」「〜です」(敬語を最小化)               |

構成ルール
-------------------------

- 結論を冒頭に書く(「〜の件、承知しました」「〜については、ご対応が難しい状況です」)
- 理由・補足・お願いの順で続ける
- 1文は60字以内を目安にする
- 1メールで伝えることは原則1〜2点に絞る

禁止事項
-------------------------

- 「お世話になっております」の多用(1通に1回まで)
- 回りくどい前置き(「突然のご連絡失礼いたします。私、〜と申しまして…」など)
- 感情的・攻撃的な表現
- 曖昧な断り方(「前向きに検討します」= 実質断りの場合は明確に書く)
- 事実と異なる内容の付け足し

出力例
-------------------------

### 例1: 納期延長の打診に同意するが条件付き(ビジネス標準トーン)

#### 素材

- 先方から2〜4週間の納期延長を打診された
- 2週間なら同意できる
- それ以上は困る

#### 出力

```
〇〇様

お世話になっております。

納期延長のご相談、承りました。
2週間程度であれば調整可能ですが、それ以上となりますと
こちらの工程に影響が出てしまうため、難しい状況です。

2週間での延長で進めていただけますでしょうか。
ご確認のほど、よろしくお願いいたします。
```

### 例2: 依頼を断る(丁寧トーン)

#### 素材

- 先方から追加作業の依頼が来た
- 今月はリソースがなく対応できない
- 来月以降なら検討できる

#### 出力

```
〇〇様

お世話になっております。

ご依頼いただきありがとうございます。
誠に恐れ入りますが、今月中の対応はリソースの都合で
お受けすることが難しい状況です。

来月以降であれば改めてご相談させていただけますと幸いです。
引き続きよろしくお願いいたします。
```

出力手順
-------------------------

1. 素材からユースケースと相手・目的を読み取る
2. トーン判定ルールに従ってトーンを決定する(ユーザー指定がある場合はそちらを優先)
3. 構成ルールに従って本文を組み立てる
4. 出力する
5. 推定したトーンや解釈を「※〇〇と解釈して処理しました」と一言添える
   (ユーザーが明示的に指定していた場合は不要)
