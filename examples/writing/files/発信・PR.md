発信・PR文作成ルール
=========================

ユーザーからSNS投稿・PR文・紹介文・説明文の依頼があった場合は、必ずこのファイルのルールに従って処理すること。

用途の判定
-------------------------

ユーザーの指示・素材から用途を判定し、対応するセクションのルールを適用する。

| 手がかり                                       | 用途    |
| ---------------------------------------------- | ------- |
| 「X」「Instagram」「Facebook」「SNS」「投稿」  | SNS投稿 |
| 「LP」「ホームページ」「商品説明」「PR文」「紹介文」 | PR文    |
| 判断できない                                   | ユーザーに確認 |

SNS投稿
-------------------------

### プラットフォーム別の基本仕様

| プラットフォーム | 文字数上限  | 推奨文字数 | ハッシュタグ | 改行         |
| ---------------- | ----------- | ---------- | ------------ | ------------ |
| X(Twitter)       | 280字(全角) | 100〜140字 | 1〜3個       | 少なめ       |
| Instagram        | 2,200字     | 150〜300字 | 5〜15個      | 読みやすく   |
| Facebook         | 制限なし    | 100〜200字 | 1〜3個(任意) | 段落で区切る |
| note             | 制限なし    | 300〜800字 | 3〜5個       | 見出しも可   |
| 指定なし         | -           | 100〜150字 | なし         | 少なめ       |

### フックの種類と使い分け

投稿の冒頭1〜2行で読者の関心を引く。以下から素材に合ったものを選ぶ。

| フックの種類 | 例                                         | 適したテーマ     |
| ------------ | ------------------------------------------ | ---------------- |
| 問いかけ     | 「〇〇で困ったこと、ありませんか?」        | 悩み・課題系     |
| 数字         | 「3つのことを意識するだけで…」             | ノウハウ・Tips系 |
| 逆説         | 「丁寧に書くほど、読まれないメールがある」 | 気づき・学び系   |
| 事実の提示   | 「先月、〇〇を始めました」                 | 近況・告知系     |
| 共感         | 「〇〇って、意外とむずかしいですよね」     | 体験談・共感系   |

### 構成ルール

X(Twitter)向け:

```
(フック: 1〜2行)
(本文: 2〜4行)
(まとめ・行動喚起: 1行)
#ハッシュタグ
```

Instagram向け:

```
(フック: 1〜2行)

(本文: 3〜6行。体験・気づき・具体例)

(まとめ・メッセージ: 1〜2行)

#ハッシュタグ #ハッシュタグ …
```

Facebook向け:

```
(フック: 1〜2行)

(本文: 2〜4行)

(締め・問いかけ: 1行)
```

### 禁止事項

- 根拠のない誇張表現(「絶対に」「100%」「業界最高」)
- 1投稿に複数のテーマを詰め込む
- 関係のないハッシュタグの大量付与
- 事実と異なる内容

### 出力例

#### 例1: X向け・サービス開始の告知(事実の提示フック)

素材:

- 来月からオンライン相談を開始する
- 初回30分無料
- 予約はDMで受け付ける

出力:

```
来月から、オンライン相談を始めます。

初回30分は無料でお試しいただけます。
「ちょっと聞いてみたい」くらいの気持ちで、
気軽にDMでご連絡ください。

#オンライン相談 #個人事業主
```

#### 例2: Instagram向け・体験談(共感フック)

素材:

- メール返信に毎回時間がかかっていた
- AIを使ったら5分で終わるようになった
- 具体的にやってること: 箇条書きで内容を投げるだけ

出力:

```
メールの返信、毎回30分かかってませんか?

私はずっとそうでした。
「何から書けばいいか」で詰まって、気づいたら1時間…

今はAIに箇条書きを投げるだけ。
5分で下書きができて、あとは微調整するだけです。

ちょっとした工夫で、じわじわ時間が返ってくる感覚があります。

#時短 #AI活用 #個人事業主 #仕事効率化
```

PR文
-------------------------

対象: ランディングページ・ホームページ・ECサイト・パンフレット・営業資料など

### 基本構成(PASBEフレームワーク)

以下の順序を基本とする。全要素が必須ではなく、素材に合わせて使う要素を選ぶ。

```
1. Problem(課題提示)     ← 読者が抱えている悩み・状況を言語化する
2. Agitation(共感)       ← 「そのままだと〇〇になる」と危機感・共感を深める(任意)
3. Solution(解決策)      ← 商品・サービスがどう解決するかを示す
4. Benefit(ベネフィット) ← 使った後にどうなるか(変化・結果)
5. Evidence(根拠)        ← 実績・数字・事例・資格など(素材にある場合)
```

素材に含まれる情報が少ない場合は、1・3・4の3要素に絞って組み立てる。

### 掲載場所別の文字数目安

| 掲載場所                       | 文字数目安 | 構成の密度 |
| ------------------------------ | ---------- | ---------- |
| LP・ホームページのメインコピー | 50〜150字  | 1〜2要素   |
| LP・ホームページのサービス説明 | 200〜400字 | 3〜4要素   |
| ECサイトの商品説明             | 150〜300字 | 3要素      |
| パンフレット・営業資料         | 300〜600字 | 4〜5要素   |
| 指定なし                       | 200〜350字 | 3〜4要素   |

### ベネフィットの書き方

「機能・特徴」をそのまま書くのではなく、「使った後の変化」として書く。

| 避けるべき書き方(機能説明)   | 望ましい書き方(ベネフィット)                     |
| ---------------------------- | ------------------------------------------------ |
| 「高速処理機能を搭載」       | 「作業時間が半分になり、定時で帰れるようになる」 |
| 「専任担当者がつく」         | 「困ったときすぐ聞けるので、迷う時間がなくなる」 |
| 「栄養バランスを考えた設計」 | 「毎日続けるうちに、体が軽くなるのを実感できる」 |

### 禁止表現

根拠のない誇張表現(素材に実績・データがある場合のみ使用可):

- 「業界No.1」「国内最大級」「圧倒的な」
- 「絶対に」「必ず」「100%」
- 「革命的な」「世界が変わる」
- 「〇〇するだけでOK」(過度に簡単に見せる表現)

検証可能な事実の捏造(素材に記載がない場合は補完しないこと):

| 捏造してはいけない表現の例                         | 素材にある場合       |
| -------------------------------------------------- | -------------------- |
| 導入実績・顧客数(「50社以上」「1,000人が利用」)    | 素材の数字のみ使用可 |
| 満足度・評価スコア(「満足度98%」「★4.8」)          | 素材の数字のみ使用可 |
| 受賞歴・認定・資格(「〇〇賞受賞」「〇〇認定」)    | 素材の記載のみ使用可 |
| メディア掲載実績(「〇〇に掲載」)                   | 素材の記載のみ使用可 |
| 創業年数・運営年数(「創業20年」)                   | 素材の記載のみ使用可 |

素材にこれらの情報がない場合は、該当箇所を空欄・省略するか、
「(実績・数字をご提供ください)」とプレースホルダーで出力すること。

### 出力例

#### 例1: 個人事業主向けオンライン相談サービス(LP説明文・200字程度)

素材:

- ターゲット: フリーランス・個人事業主
- 悩み: 税務・契約・経営の判断を一人でしなければならない
- サービス: オンライン相談(30分単位)
- 特徴: 当日予約可・業務経験10年以上の専門家が対応

出力:

```
一人で抱え込まなくていい。

フリーランスや個人事業主は、税務・契約・経営判断を
すべて自分でこなさなければなりません。
「これで合っているのか」という不安を抱えたまま
決断し続けるのは、じわじわと体力を消耗します。

当サービスは、業務経験10年以上の専門家に
30分単位でオンライン相談できるサービスです。
当日予約も可能なので、「今すぐ聞きたい」に対応できます。

判断に迷ったとき、すぐ相談できる環境があるだけで
仕事の進み方が変わります。
```

#### 例2: ハンドメイド商品のECサイト説明文(150字程度)

素材:

- 商品: 天然石を使ったブレスレット
- 特徴: 一点物・受注生産・天然石は国内仕入れ
- ターゲット: 自分へのご褒美を探している人

出力:

```
世界に一つだけの、自分だけのブレスレット。

すべて受注生産の一点物です。
国内で厳選した天然石を使用し、ご注文をいただいてから
一つひとつ手作りしています。

「自分へのご褒美に」「記念日に」——
そんなときの特別な一本として、お選びいただいています。
```

出力手順
-------------------------

1. 素材・指示から用途(SNS投稿/PR文)を判定する
2. SNS投稿の場合: プラットフォームを確認し、文字数・フック・構成ルールを適用する
3. PR文の場合: 掲載場所を確認し、PASBEの使用要素と文字数目安を決定する
4. 検証可能な事実(数字・実績・受賞歴など)が素材にあるか確認する
5. 素材にない検証可能な事実を補完していないか確認する
6. 禁止表現が含まれていないか確認する
7. 出力する
8. 推定した用途・プラットフォームや使用したフレームワーク要素を添える
   (ユーザーが明示的に指定していた場合は不要)
