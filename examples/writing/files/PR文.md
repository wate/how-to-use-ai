PR文ルール
=========================

ユーザーからPR文・紹介文・説明文の依頼があった場合は、必ずこのファイルのルールに従って処理すること。
対象: ランディングページ・ホームページ・ECサイト・パンフレット・営業資料など

基本構成(PASBEフレームワーク)
-------------------------

以下の順序を基本とする。全要素が必須ではなく、素材に合わせて使う要素を選ぶ。

```
1. Problem(課題提示)  ← 読者が抱えている悩み・状況を言語化する
2. Agitation(共感)    ← 「そのままだと〇〇になる」と危機感・共感を深める(任意)
3. Solution(解決策)   ← 商品・サービスがどう解決するかを示す
4. Benefit(ベネフィット) ← 使った後にどうなるか(変化・結果)
5. Evidence(根拠)     ← 実績・数字・事例・資格など(素材にある場合)
```

素材に含まれる情報が少ない場合は、1・3・4の3要素に絞って組み立てる。

掲載場所別の文字数目安
-------------------------

| 掲載場所                       | 文字数目安 | 構成の密度 |
| ------------------------------ | ---------- | ---------- |
| LP・ホームページのメインコピー | 50〜150字  | 1〜2要素   |
| LP・ホームページのサービス説明 | 200〜400字 | 3〜4要素   |
| ECサイトの商品説明             | 150〜300字 | 3要素      |
| パンフレット・営業資料         | 300〜600字 | 4〜5要素   |
| 指定なし                       | 200〜350字 | 3〜4要素   |

ベネフィットの書き方
-------------------------

「機能・特徴」をそのまま書くのではなく、「使った後の変化」として書く。

| 避けるべき書き方(機能説明)   | 望ましい書き方(ベネフィット)                     |
| ---------------------------- | ------------------------------------------------ |
| 「高速処理機能を搭載」       | 「作業時間が半分になり、定時で帰れるようになる」 |
| 「専任担当者がつく」         | 「困ったときすぐ聞けるので、迷う時間がなくなる」 |
| 「栄養バランスを考えた設計」 | 「毎日続けるうちに、体が軽くなるのを実感できる」 |

禁止表現
-------------------------

### 根拠のない誇張表現

以下は根拠が示せない場合に使用禁止。素材に実績・データがある場合のみ使用可。

- 「業界No.1」「国内最大級」「圧倒的な」
- 「絶対に」「必ず」「100%」
- 「革命的な」「世界が変わる」
- 「〇〇するだけでOK」(過度に簡単に見せる表現)

### 検証可能な事実の捏造

以下は「事実の形をとる表現」であるため、素材に記載がない場合は補完しないこと。
形容詞的な誇張表現(「こだわり抜いた」「丁寧な仕上がり」など)とは区別する。

| 捏造してはいけない表現の例   | 素材にある場合   |
| ---------------------------- | ---------------- |
| 導入実績・顧客数(「50社以上」「1,000人が利用」) | 素材の数字のみ使用可 |
| 満足度・評価スコア(「満足度98%」「★4.8」)       | 素材の数字のみ使用可 |
| 受賞歴・認定・資格(「〇〇賞受賞」「〇〇認定」) | 素材の記載のみ使用可 |
| メディア掲載実績(「〇〇に掲載」)               | 素材の記載のみ使用可 |
| 創業年数・運営年数(「創業20年」)               | 素材の記載のみ使用可 |

素材にこれらの情報がない場合は、該当箇所を空欄・省略するか、
「(実績・数字をご提供ください)」とプレースホルダーで出力すること。

出力例
-------------------------

### 例1: 個人事業主向けオンライン相談サービス(LP説明文・200字程度)

#### 素材

- ターゲット: フリーランス・個人事業主
- 悩み: 税務・契約・経営の判断を一人でしなければならない
- サービス: オンライン相談(30分単位)
- 特徴: 当日予約可・業務経験10年以上の専門家が対応

#### 出力

```
一人で抱え込まなくていい。

フリーランスや個人事業主は、税務・契約・経営判断を
すべて自分でこなさなければなりません。
「これで合っているのか」という不安を抱えたまま
決断し続けるのは、じわじわと体力を消耗します。

当サービスは、業務経験10年以上の専門家に
30分単位でオンライン相談できるサービスです。
当日予約も可能なので、「今すぐ聞きたい」に対応できます。

判断に迷ったとき、すぐ相談できる環境があるだけで
仕事の進み方が変わります。
```

### 例2: ハンドメイド商品のECサイト説明文(150字程度)

#### 素材

- 商品: 天然石を使ったブレスレット
- 特徴: 一点物・受注生産・天然石は国内仕入れ
- ターゲット: 自分へのご褒美を探している人

#### 出力

```
世界に一つだけの、自分だけのブレスレット。

すべて受注生産の一点物です。
国内で厳選した天然石を使用し、ご注文をいただいてから
一つひとつ手作りしています。

「自分へのご褒美に」「記念日に」——
そんなときの特別な一本として、お選びいただいています。
```

出力手順
-------------------------

1. 素材から商品・サービスの特徴・ターゲット・掲載場所を読み取る
2. 掲載場所別の文字数目安を確認する(指定がない場合は200〜350字)
3. 素材の情報量に応じてPASBEの使用要素を選ぶ
4. ベネフィットの書き方に従い「機能説明」を「変化の描写」に変換する
5. 検証可能な事実(数字・実績・受賞歴など)が素材にあるか確認する
6. 素材にない検証可能な事実を補完していないか確認する
7. 禁止表現が含まれていないか確認する
8. 出力する
9. 使用したフレームワーク要素と文字数を添える
