文章作成
=========================

メール返信・SNS投稿・PR文・ブログ記事など、日常的な文章作成を
最小限の入力で済むようにする活用事例です。

素材(箇条書き・メモ・下書き)を投げるだけで、登録したファイルのルールに従って
AIが構成・トーン・文字数を自動で判断して出力します。

対象プラットフォーム
-------------------------

| プラットフォーム | 想定プラン     | モデル          |
| ---------------- | -------------- | --------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.3 Instant |
| Gemini           | 無料プラン以上 | Gemini 3 Flash  |

ファイル構成
-------------------------

| ファイル                  | 役割                                         |
| ------------------------- | -------------------------------------------- |
| `カスタム指示.md`         | カスタム指示欄に貼り付ける内容               |
| `files/メール返信.md`     | メール返信の構成・トーン・出力例             |
| `files/チャット返信.md`   | チャット返信の簡潔さの基準・出力例           |
| `files/SNS投稿.md`        | SNS投稿の文字数・フック・ハッシュタグ・出力例 |
| `files/PR文.md`           | PR文の構成・禁止表現・出力例                 |
| `files/ブログ記事.md`     | ブログ記事の見出し構成・導入パターン・出力例 |
| `files/要約・変換.md`     | 要約・言い換え・トーン変換の各ルール・出力例 |

どのファイルを登録するか
-------------------------

ChatGPT 無料プランはプロジェクトへの登録ファイルが **5つまで** です。
全ファイルを登録する必要はありません。以下の棚卸し表を参考に、
自分が使うユースケースのファイルだけ選んでください。

### ユースケース棚卸し表

| 自分の用途                                   | 登録するファイル        |
| -------------------------------------------- | ----------------------- |
| 取引先・顧客へのメール返信                   | `メール返信.md`         |
| 社内チャット・Slack・LINE での返信           | `チャット返信.md`       |
| X(Twitter)・Instagram・Facebook への投稿     | `SNS投稿.md`            |
| 商品・サービスの紹介文・ランディングページ   | `PR文.md`               |
| 自社ブログ・note・オウンドメディアへの投稿   | `ブログ記事.md`         |
| 長い文章の要約・言い換え・トーン変換         | `要約・変換.md`         |

#### 例: 個人事業主(顧客対応＋ホームページ運営)の場合

```
登録するファイル:
- メール返信.md    ← 顧客とのやり取りがメール中心
- PR文.md          ← 商品・サービスページの紹介文
- ブログ記事.md    ← 活用事例や情報発信

計3ファイル → 無料プランでも予備枠が2つ残る
```

セットアップ(初回のみ)
-------------------------

### ChatGPT(プロジェクト機能)

1. プロジェクトを作成する(例: 「文章作成」)
2. `カスタム指示.md` の内容をプロジェクトのカスタム指示欄に貼り付ける
3. 上の棚卸し表で選んだファイルを情報源として登録する

### Gemini(Gems 機能)

1. Gem を作成する(例: 「文章作成」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 上の棚卸し表で選んだファイルをナレッジファイルとして登録する

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# メール返信を作る場合
以下の内容でメール返信を作って

- 先方からの納期延長の打診に同意する
- ただし2週間以上の延長は困る旨を伝えたい
- 丁寧だが率直なトーンで

# SNS投稿を作る場合
新しいサービスの告知投稿を作って(X向け)

- 来月からオンライン相談を開始する
- 初回30分無料
- 予約はDMで受け付ける

# 既存の文章をリライトする場合
このメールをもっと簡潔にして(文章を貼り付け)
```

チャット管理
-------------------------

- 用途ごとにチャットを分けて管理する
    - 例: 「メール返信 2026年5月」「ブログ下書き 2026年5月」
- チャットが長くなったら新チャットに切り替える
- 前回と同じ用途なら、最初の1行で「メール返信で」と添えるだけで引き継げる
