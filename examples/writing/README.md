文章作成
=========================

メール返信・SNS投稿・PR文・ブログ記事など、日常的な文章作成を
最小限の入力で済むようにする活用事例です。

素材(箇条書き・メモ・下書き)を投げるだけで、登録したファイルのルールに従って
AIが構成・トーン・文字数を自動で判断して出力します。

対象プラットフォーム
-------------------------

| プラットフォーム | 想定プラン     | モデル            |
| ---------------- | -------------- | ----------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.5 Instant   |
| Claude           | 無料プラン以上 | Claude Sonnet 4.6 |

セットアップ(初回のみ)
-------------------------

> プロジェクト の概要や共通の考え方は [活用事例 README](../README.md) を参照してください。

### 基本的な流れ

1. [プロジェクトを作成](../README.md#プロジェクト機能とは)する(例: 「文章作成」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 登録対象ファイルを各プラットフォームの指定の項目に登録する
    - ChatGPT: 「情報源」項目
    - Claude: PC版は「ファイル」、スマートフォン版は「プロジェクトナレッジ」

### 登録対象ファイル

ここでの登録対象ファイルはfiles/以下です。

- [files/コミュニケーション.md](files/コミュニケーション.md)
- [files/発信・PR.md](files/発信・PR.md)
- [files/ブログ記事.md](files/ブログ記事.md)
- [files/要約・変換.md](files/要約・変換.md)

プロフィールを作成した場合は、必要に応じて追加で登録してください。
プライベートか仕事かの文脈判断が正確になります。

- 推奨: プロジェクトにファイルとして登録する(プロジェクト単位で文脈を変えたい場合)
- フォールバック: アカウントのプロファイル設定に登録する(全チャットで自動適用、プロジェクトにファイルがない場合に使用される)

作り方は [プロフィール生成プロンプト](../../prompts/プロフィール生成.md) を参照してください。

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# メール返信を作る場合
以下の内容でメール返信を作って

- 先方からの納期延長の打診に同意する
- ただし2週間以上の延長は困る旨を伝えたい
- 丁寧だが率直なトーンで

# SNS投稿を作る場合(プロフィールあり)
新商品のX向け告知投稿を作って
※ プロフィールの情報を参照して処理される

# 既存の文章をリライトする場合
このメールをもっと簡潔にして(文章を貼り付け)
```

その他の活用事例は[活用事例](#活用事例)を参照。

チャット管理
-------------------------

- 用途ごとにチャットを分けて管理する
    - 例: 「メール返信 2026年5月」「ブログ下書き 2026年5月」
- チャットが長くなったら新チャットに切り替える
- 前回と同じ用途なら、最初の1行で「メール返信で」と添えるだけで引き継げる

活用事例
-------------------------

- メール返信: 問い合わせ・依頼・クレームへの返信文を素材から作る
- SNS投稿: X・Instagramへの告知・近況投稿を箇条書きから作る
- ブログ記事: メモや下書きから構成を整えた記事に仕上げる
- リライト: 既存の文章をターゲット・トーンに合わせて書き直す

### 事業者向け活用例

- 問い合わせ対応: 顧客からの問い合わせメールへの返信文を素材から作る
- お礼・案内メール: 訪問後のお礼・サービス案内・見積もり送付時の添え文を作る
- 提案・見積もりの説明文: 見積書に添える説明文や提案の要点をまとめた文章を作る
- 告知文: 新サービスや期間限定対応をSNS・チラシ向けに告知する文章を作る
