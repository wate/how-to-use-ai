活用事例
=========================

このリポジトリでは、ChatGPT・Gemini・Claude を使った日常の活用事例をまとめています。  
プロンプトを毎回入力するのではなく、**一度設定しておけばファイルを添付するだけで使える**構成を目指しています。

考え方
-------------------------

AI 活用の解説書やブログでは、プロンプトの書き方が中心に説明されています。  
しかし、毎回同じプロンプトを打ち込むのは手間です。

そこで、このリポジトリでは以下の構成を基本としています。

- **カスタム指示**：AIに「いつも守ってほしいルール」を事前に登録しておく(例: 丁寧な文体で回答する)
- **情報源ファイル**：AIに参照させたい資料ファイルを登録しておく(例: 社内ルール、商品一覧、過去のQ&A)

この2つを各サービスの機能(ChatGPT/Claudeのプロジェクト、GeminiのGems)に設定しておくことで、日常利用時はファイルを添付するだけで済みます。

プロジェクト / Gems とは
-------------------------

### ChatGPT のプロジェクト機能

関連するチャット・ファイル・カスタム指示を一箇所にまとめて管理できる機能です。  
プロジェクトを選択してチャットを始めると、登録したカスタム指示と情報源ファイルが自動的に参照されます。  
無料プランから利用可能です。

### Gemini の Gems 機能

独自の指示・役割設定(どんな立場で答えるか)・参照資料ファイルを設定した、再利用可能なカスタム AI アシスタントを作成できる機能です。  
無料プランでも作成可能です(作成数・利用回数に制限あり)。

### Claude のプロジェクト機能

関連するチャット・ファイル・カスタム指示を一箇所にまとめて管理できる機能です。  
プロジェクトを選択してチャットを始めると、登録したカスタム指示と情報源ファイルが自動的に参照されます。  
無料プランから利用可能ですが、作成できるプロジェクトは **5つまで** です。

基本的な構成
-------------------------

各事例は以下のファイルで構成されています。

| ファイル             | 役割                                                       |
| -------------------- | ---------------------------------------------------------- |
| `カスタム指示.md`    | プロジェクト / Gems のカスタム指示欄に貼り付ける内容       |
| 情報源ファイル(複数) | ルール定義・データなど。プロジェクト / Gems に登録しておく |

ChatGPT 無料プランはプロジェクトへの登録ファイルが **5つまで** です。
Claude 無料プランは作成できるプロジェクトが **5つまで** です。
いずれも全ファイルを登録する必要はなく、自分がよく使う用途のファイルだけ選んでください。
各事例の README にユースケース棚卸し表を用意しています。

使い方の流れ
-------------------------

### セットアップ(初回のみ)

1. ChatGPT でプロジェクトを作成する(または Gemini で Gem を、Claude でプロジェクトを作成する)
2. `カスタム指示.md` の内容をカスタム指示欄に貼り付ける
3. 情報源ファイルを登録する

各事例の README にプラットフォームごとの詳細手順を記載しています。

### 事前準備(オプション)

プロフィールを設定しておくと、各事例で共通して自分の情報を参照させることができます。
AIがヒアリングしながらファイルを生成してくれるため、手書きでの作成は不要です。

使い方は以下のいずれかです。

1. **プロジェクト/Gemsにファイルとして登録する(推奨)**
    - プロジェクト単位で文脈を変えたい場合に有効
    - プロジェクトに登録されている場合、カスタム指示から優先して参照される
2. **アカウントのプロファイル設定に登録する**
    - 全チャットで自動的に適用される
    - プロジェクトにファイルがない場合のフォールバックとして機能する

詳しくは [プロフィール生成プロンプト](../prompts/プロフィール生成.md) を参照してください。

### 日常利用

セットアップ後は、以下の流れで使えます。

```
1. ChatGPT/Claudeではプロジェクト、GeminiではGemsを選択してチャットを開始する
2. 以下のいずれかの方法で送信する
   - ファイルや画像を添付して送信する(例：レシート画像を添付するだけ)
   - 最低限の指示を入力して送信する(例：「このメールの返信文を考えて」)
3. 必要に応じて出力形式などを個別に指定する
   (例：「freee で取り込める形式で出して」)
```

カスタム指示に動作ルールが仕込まれているため、毎回細かい指定をする必要はありません。

### チャット管理

- 用途ごとにチャットを分けて管理する
- チャットが長くなったら新チャットに切り替える
- 前回の続きから始めたい場合は、最初に状況を簡単に補足する

セキュリティと情報の取り扱い
-------------------------

ChatGPT・Gemini・Claudeのデータ利用方針は、プラン・設定・時期によって異なります。
利用前に各プラットフォームの公式ポリシーを確認してください。
以下の点に注意して利用してください。

- 本名・住所・電話番号など個人を特定できる情報は入力しない
- 社外秘・機密情報は入力しない
- 学習への利用設定や履歴設定を確認し、必要に応じて見直す(設定方法は各プラットフォームのドキュメントを参照)

> 法人向けプランでも契約や管理設定によって挙動が異なる場合があります。運用前に管理者向けドキュメントを確認してください。
