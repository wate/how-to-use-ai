お金の管理 設計資料
=========================

概要
-------------------------

レシート画像やカード明細などを会計ソフト・家計簿サービス・Excel/スプレッドシートに
取り込みやすいCSV形式に変換するプロンプト一式の設計資料。

### 対象プラットフォームと想定プラン

| プラットフォーム | 想定プラン     | モデル          | 備考                                             |
| ---------------- | -------------- | --------------- | ------------------------------------------------ |
| ChatGPT          | **無料プラン** | GPT-5.5 Instant | プロジェクト機能・画像添付が利用可               |

> 有料プランでも利用可能だが、無料プランで動作することを前提に設計する。

要件
-------------------------

### 目的

- レシート画像・カード明細からCSVへの変換作業の手間を減らす
- 取り込み先ソフトへの実際の登録作業はユーザーに委ねる(スコープ外)

### 対応用途

| 用途   | 対象ユーザー           |
| ------ | ---------------------- |
| 家計簿 | 個人・家庭での支出管理 |
| 会計   | 個人事業主の経費管理   |

### 対応フォーマット

#### 家計簿用

| 出力形式の指定   | 出力フォーマット        | 備考                               |
| ---------------- | ----------------------- | ---------------------------------- |
| 指定なし         | 汎用CSV                 | Excel・スプレッドシート向け        |
| マネーフォワード | マネーフォワード ME形式 |                                    |
| Zaim             | Zaim形式                | ⚠️ サポート対象外のため動作保証なし |

#### 会計用

| 出力形式の指定   | 出力フォーマット              | 備考                        |
| ---------------- | ----------------------------- | --------------------------- |
| 指定なし         | 汎用CSV                       | Excel・スプレッドシート向け |
| freee            | freee形式                     |                             |
| マネーフォワード | マネーフォワード クラウド形式 |                             |
| 弥生             | 弥生会計形式                  |                             |

-------------------------

```

│   └ カスタム指示.md の内容を貼り付け
├ 情報源ファイル①：レシート変換ルール-家計簿.md

├ 情報源ファイル③：勘定科目振り分けデータ-家計簿.csv(要カスタマイズ)
└ 情報源ファイル④：勘定科目振り分けデータ-会計.csv(要カスタマイズ)
```

ファイル仕様
-------------------------

### カスタム指示.md

プロジェクトのカスタム指示欄に貼り付ける内容。
デフォルトの用途・出力形式と、情報源ファイルの参照ルールを定義する。

- デフォルト用途：家計簿
- デフォルト出力形式：汎用CSV
- ユーザー指定がある場合はそちらを優先

### レシート変換ルール-家計簿.md / レシート変換ルール-会計.md

各用途のフォーマット定義・費目判定ルール・読み取りルール・出力手順を定義する。

#### 共通の読み取りルール

- 日付：YYYY-MM-DD形式(弥生のみ YYYY/MM/DD)
- 金額：税込合計を使用
- 消費税額が未記載の場合は逆算(10%: 税込÷1.1×0.1、8%: 税込÷1.08×0.08、端数切り捨て)
- 税区分混在(8%・10%)：行を分けて出力
- 読み取り不能項目：`(判別不能)` と記載
- 複数枚のレシート：それぞれ別行で出力

### 勘定科目振り分けデータ-家計簿.csv

```
取引先(店名),費目(品目),費目,内訳
```

- 取引先または費目(品目)のどちらか一方だけの指定も可
- 費目：大項目(食費・日用品など)
- 内訳：中項目(食料品・外食など)
- 一致するデータがない場合は自動推定にフォールバック

### 勘定科目振り分けデータ-会計.csv

```
取引先(店名),費目(品目),勘定科目
```

- 取引先または費目(品目)のどちらか一方だけの指定も可
- 一致するデータがない場合は自動推定にフォールバック

設計判断ログ
-------------------------

| 判断                           | 理由                                                                 |
| ------------------------------ | -------------------------------------------------------------------- |
| プロジェクト機能を使う         | カスタム指示がChatGPT全体に影響するのを避けるため                    |
| 情報源ファイルを用途ごとに分割 | ルールを分離し保守しやすくするため                                   |
| 勘定科目データをCSVで管理      | Excelで編集しやすく、読み取り精度も安定するため                      |
| 取り込み先の操作はスコープ外   | ソフトの種類・バージョンが多岐にわたりプロンプトで対応しきれないため |
| Zaimは動作保証対象外と注記     | 2019年8月にCSVアップロードのサポートが終了しているため               |
