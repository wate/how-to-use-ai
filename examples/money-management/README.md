お金の管理
=========================

レシート画像やカード明細を、会計ソフト・家計簿サービス・Excel に取り込みやすい  
CSV 形式に変換する活用事例です。

対象プラットフォーム
-------------------------

| プラットフォーム | 想定プラン     | モデル          |
| ---------------- | -------------- | --------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.3 Instant |
| Gemini           | 無料プラン以上 | Gemini 3 Flash  |

ファイル構成
-------------------------

| ファイル                                  | 役割                                     |
| ----------------------------------------- | ---------------------------------------- |
| `カスタム指示.md`                         | カスタム指示欄に貼り付ける内容           |
| `files/レシート変換ルール-家計簿.md`      | 家計簿用フォーマット・ルール定義         |
| `files/レシート変換ルール-会計.md`        | 会計用フォーマット・ルール定義           |
| `files/勘定科目振り分けデータ-家計簿.csv` | 家計簿用費目マッピング(要カスタマイズ)   |
| `files/勘定科目振り分けデータ-会計.csv`   | 会計用勘定科目マッピング(要カスタマイズ) |

セットアップ(初回のみ)
-------------------------

### ChatGPT(プロジェクト機能)

1. プロジェクトを作成する（例：「家計簿・経費入力」）
2. `カスタム指示.md` の内容をプロジェクトのカスタム指示欄に貼り付ける
3. `files/` 内の4ファイルを情報源として登録する
4. `勘定科目振り分けデータ-*.csv` のサンプル行を実際の取引先・ルールに書き換える

### Gemini(Gems 機能)

1. Gem を作成する（例：「家計簿・経費入力」）
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. `files/` 内の4ファイルをナレッジファイルとして登録する
4. `勘定科目振り分けデータ-*.csv` のサンプル行を実際の取引先・ルールに書き換える

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# デフォルト設定のまま使う場合(家計簿・汎用CSV)
レシート画像を添付して送信するだけ

# 出力形式を指定する場合
マネーフォワードで取り込めるよう変換して(画像添付)

# 用途と出力形式を両方指定する場合
会計用で freee に変換して(画像添付)
```

チャット管理
-------------------------

- 用途ごとにチャットを分けて管理する
    - 例：「家計簿 2026年5月」「経費 2026年5月」
- チャットが長くなったら新チャットに切り替え、最初の1回だけ用途・出力形式を再指定する
- カスタム指示のデフォルトを自分がよく使う設定にしておけば、再指定も不要
