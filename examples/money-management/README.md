お金の管理
=========================

レシート画像やカード明細を、会計ソフト・家計簿サービス・Excelに取り込みやすいCSV形式に変換する活用事例です。

対象プラットフォーム
-------------------------

| プラットフォーム | 想定プラン     | モデル            |
| ---------------- | -------------- | ----------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.5 Instant   |
| Gemini           | 無料プラン以上 | Gemini 3 Flash    |
| Claude           | 無料プラン以上 | Claude Sonnet 4.6 |

ファイル構成
-------------------------

| ファイル                                                                           | 役割                                     |
| ---------------------------------------------------------------------------------- | ---------------------------------------- |
| [カスタム指示.md](カスタム指示.md)                                                 | カスタム指示欄に貼り付ける内容           |
| [files/レシート変換ルール-家計簿.md](files/レシート変換ルール-家計簿.md)           | 家計簿用フォーマット・ルール定義         |
| [files/レシート変換ルール-会計.md](files/レシート変換ルール-会計.md)               | 会計用フォーマット・ルール定義           |
| [files/勘定科目振り分けデータ-家計簿.csv](files/勘定科目振り分けデータ-家計簿.csv) | 家計簿用費目マッピング(要カスタマイズ)   |
| [files/勘定科目振り分けデータ-会計.csv](files/勘定科目振り分けデータ-会計.csv)     | 会計用勘定科目マッピング(要カスタマイズ) |

セットアップ(初回のみ)
-------------------------

> プロジェクト / Gems の概要や共通の考え方は [活用事例 README](../README.md) を参照してください。

### 基本的な流れ

1. [プロジェクト/Gems を作成](../README.md#プロジェクト-gemsとは)する(例: 「家計簿・経費入力」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 登録対象ファイルを各プラットフォームの指定の項目に登録する
   - ChatGPT: 「情報源」項目
   - Gemini: 「知識」項目
   - Claude: 「ファイル」項目
4. データのカスタマイズを行う

### 登録対象ファイル

ここでの登録対象ファイルはfiles/以下です。

- [files/レシート変換ルール-家計簿.md](files/レシート変換ルール-家計簿.md)
- [files/レシート変換ルール-会計.md](files/レシート変換ルール-会計.md)
- [files/勘定科目振り分けデータ-家計簿.csv](files/勘定科目振り分けデータ-家計簿.csv)
- [files/勘定科目振り分けデータ-会計.csv](files/勘定科目振り分けデータ-会計.csv)

プロフィールを作成した場合は、必要に応じて追加で登録してください。
プライベートか仕事かの文脈判断が正確になります。

- 推奨: プロジェクト/Gemsにファイルとして登録する(プロジェクト単位で文脈を変えたい場合)
- フォールバック: アカウントのプロファイル設定に登録する(全チャットで自動適用、プロジェクトにファイルがない場合に使用される)

作り方は [プロフィール生成プロンプト](../../prompts/プロフィール生成.md) を参照してください。

`勘定科目振り分けデータ-*.csv` のサンプル行を実際の取引先・ルールに書き換えてください。勘定科目データのたたき台を生成する方法は[Tips](#tips) を参照してください。

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# デフォルト設定のまま使う場合(家計簿・汎用CSV)
レシート画像を添付して送信するだけ

# 出力形式を指定する場合
マネーフォワードで取り込めるよう変換して(画像添付)

# 用途と出力形式を両方指定する場合
会計用で freee に変換して(画像添付)
```

チャット管理
-------------------------

- 用途ごとにチャットを分けて管理する
    - 例：「家計簿 2026年5月」「経費 2026年5月」
- チャットが長くなったら新チャットに切り替え、最初の1回だけ用途・出力形式を再指定する
- カスタム指示のデフォルトを自分がよく使う設定にしておけば、再指定も不要

Tips
-------------------------

### 勘定科目データのたたき台を作る

過去のレシート画像をまとめて添付し、以下のプロンプトを送信すると勘定科目データのたたき台を生成できます。
生成結果は必ず確認・修正した上で登録してください。

#### 家計簿用

````
これらのレシートをもとに、以下のフォーマットで取引先と費目の一覧を作成して

```
取引先(店名),費目(品目),費目,内訳
```
````

#### 会計用

````
これらのレシートをもとに、以下のフォーマットで取引先と勘定科目の一覧を作成して

```
取引先(店名),費目(品目),勘定科目
```
````
