レシート変換ルール(会計用)
=========================

ユーザーから変換依頼があった場合は、必ずこのファイルのルールに従って処理してください。
ソフト名が指定された場合は対応フォーマットで、指定がない場合は汎用CSVで出力してください。

フォーマット定義
-------------------------

### 汎用CSV(出力形式の指定なし・またはExcel/スプレッドシート向け)

列順: 日付,金額(税込),消費税額,税区分,取引先,品目/摘要,勘定科目,備考

出力例:

```
日付,金額(税込),消費税額,税区分,取引先,品目/摘要,勘定科目,備考
2024-06-01,1100,100,10%,〇〇スーパー,食料品,消耗品費,
2024-06-01,540,40,8%,〇〇スーパー,飲料,消耗品費,
```

### freee(出力形式: freee)

列順: 取引日,収支区分,取引先,品目,税区分,金額,消費税額,勘定科目,メモ

出力例:

```
取引日,収支区分,取引先,品目,税区分,金額,消費税額,勘定科目,メモ
2024-06-01,支出,〇〇スーパー,食料品,課税仕入10%,1100,100,消耗品費,
```

### マネーフォワード クラウド(出力形式: マネーフォワード)

列順: 決算日,借方勘定科目,借方税区分,借方金額,借方税額,貸方勘定科目,貸方税区分,貸方金額,貸方税額,摘要,メモ,タグ,MF仕訳ID

出力例:

```
決算日,借方勘定科目,借方税区分,借方金額,借方税額,貸方勘定科目,貸方税区分,貸方金額,貸方税額,摘要,メモ,タグ,MF仕訳ID
2024-06-01,消耗品費,課税仕入10%,1100,100,現金,対象外,1100,,〇〇スーパー 食料品,,,
```

### 弥生会計(出力形式: 弥生)

列順: 伝票日付,借方科目,借方補助科目,借方税区分,借方金額,借方消費税額,貸方科目,貸方補助科目,貸方税区分,貸方金額,貸方消費税額,摘要,番号,期日,メモ

出力例:

```
伝票日付,借方科目,借方補助科目,借方税区分,借方金額,借方消費税額,貸方科目,貸方補助科目,貸方税区分,貸方金額,貸方消費税額,摘要,番号,期日,メモ
2024/06/01,消耗品費,,課税仕入10%,1100,100,現金,,,1100,,〇〇スーパー 食料品,,,
```

勘定科目の判定ルール
-------------------------

勘定科目は以下の優先順位で決定すること。

1. 「勘定科目振り分けデータ-会計.csv」が情報源として登録されている場合 → そのデータに従う
2. 該当するデータがない場合 → 以下の目安で自動推定する

| 品目・店舗の特徴               | 勘定科目   |
| ------------------------------ | ---------- |
| スーパー・コンビニ(食品・飲料) | 消耗品費   |
| コンビニ(コピー・印刷)         | 消耗品費   |
| 文具・事務用品                 | 消耗品費   |
| 外食・カフェ(接待目的)         | 交際費     |
| 外食・カフェ(社内打ち合わせ)   | 会議費     |
| 書籍                           | 新聞図書費 |
| 交通機関・バス・タクシー       | 旅費交通費 |
| ガソリンスタンド               | 車両費     |
| 宿泊施設                       | 旅費交通費 |
| 医療・薬局                     | 福利厚生費 |
| 通信・ネット                   | 通信費     |
| その他・不明                   | 雑費       |

読み取りルール
-------------------------

- 日付は YYYY-MM-DD 形式で出力する(弥生のみ YYYY/MM/DD 形式)
- 金額は税込合計を使用する
- 消費税額が明記されていない場合は税込金額から逆算する
    - 10%: 税込 ÷ 1.1 × 0.1(端数切り捨て)
    - 8%: 税込 ÷ 1.08 × 0.08(端数切り捨て)
- 税区分が混在する場合(8%・10%両方)は行を分けて出力する
- 読み取れない項目は `(判別不能)` と記載する
- レシートが複数枚ある場合は、それぞれ別行で出力する

出力手順
-------------------------

1. 画像を確認し、以下の項目を読み取る
    - 日付 / 取引先 / 品目 / 金額 / 消費税額 / 税区分
2. 出力形式を以下の優先順位で決定する
    1. 今回のメッセージで出力形式の指定がある → その形式を使う
    2. 今回の指定がない → チャット履歴から前回の出力形式を引き継ぐ
    3. 履歴にも指定がない → カスタム指示のデフォルト設定を使う
3. CSVをコードブロック内に出力する
4. 読み取り不確実な箇所がある場合は、CSV出力後に「※確認事項」として列挙する
