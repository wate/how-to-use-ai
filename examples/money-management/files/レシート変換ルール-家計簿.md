レシート変換ルール(家計簿用)
=========================

ユーザーから変換依頼があった場合は、必ずこのファイルのルールに従って処理してください。
ソフト名が指定された場合は対応フォーマットで、指定がない場合は汎用CSVで出力してください。

フォーマット定義
-------------------------

### 汎用CSV(出力形式の指定なし・またはExcel/スプレッドシート向け)

列順: 日付,金額,費目,内訳,お店/支払先,メモ

出力例:

```
日付,金額,費目,内訳,お店/支払先,メモ
2024-06-01,1100,食費,食料品,〇〇スーパー,
2024-06-01,540,食費,飲料,〇〇スーパー,
```

### マネーフォワード ME(出力形式: マネーフォワード)

列順: 計算対象,日付,内容,金額(円),保有金融機関,大項目,中項目,メモ

- 計算対象: 1(対象)または 0(対象外)。通常は 1
- 保有金融機関: 現金の場合は「現金」と記載

出力例:

```
計算対象,日付,内容,金額(円),保有金融機関,大項目,中項目,メモ
1,2024-06-01,〇〇スーパー 食料品,-1100,現金,食費,食料品,
```

### Zaim(出力形式: Zaim)

> ⚠️ ZaimのCSVアップロード機能はサポート対象外です。取り込めない場合は汎用CSVをご利用ください。

列順: 日付,方法,カテゴリ,カテゴリの内訳,支払元,入金先,品目,メモ,お店,支出,収入

- 方法: 支出の場合は「payment」、収入は「income」
- 支払元: 現金の場合は「財布」

出力例:

```
日付,方法,カテゴリ,カテゴリの内訳,支払元,入金先,品目,メモ,お店,支出,収入
2024-06-01,payment,食費,食料品,財布,,食料品,,〇〇スーパー,1100,0
```

費目の判定ルール
-------------------------

費目(勘定科目)は以下の優先順位で決定すること。

1. 「勘定科目振り分けデータ-家計簿.csv」が情報源として登録されている場合 → そのデータに従う
2. 該当するデータがない場合 → 以下の目安で自動推定する

| 品目・店舗の特徴               | 費目(大項目) | 内訳(中項目)   |
| ------------------------------ | ------------ | -------------- |
| スーパー・コンビニ(食品・飲料) | 食費         | 食料品         |
| 外食・カフェ・ファストフード   | 食費         | 外食           |
| ドラッグストア・薬局           | 日用品       | 医薬品         |
| コンビニ(日用品・雑貨)         | 日用品       | 日用雑貨       |
| コンビニ(コピー・印刷)         | 日用品       | その他         |
| 衣料品・ファッション           | 衣服・美容   | 衣服           |
| 美容院・理容室                 | 衣服・美容   | 美容・理容     |
| 交通機関・バス・タクシー       | 交通費       | 交通費         |
| ガソリンスタンド               | 車・バイク   | ガソリン       |
| 書籍・本                       | 教養・教育   | 書籍           |
| 医療機関・クリニック           | 医療・保険   | 医療費         |
| 電気・ガス・水道               | 水道・光熱費 | (種別に応じて) |
| 通信・ネット                   | 通信費       | 通信費         |
| 娯楽・レジャー                 | 娯楽・趣味   | (内容に応じて) |
| その他・不明                   | その他       | その他         |

読み取りルール
-------------------------

- 日付は YYYY-MM-DD 形式で出力する
- 金額は税込合計を使用する(マイナス表記不要。ただしマネーフォワードMEは支出をマイナスで記載)
- 消費税額が明記されていない場合は税込金額から逆算する
    - 10%: 税込 ÷ 1.1 × 0.1(端数切り捨て)
    - 8%: 税込 ÷ 1.08 × 0.08(端数切り捨て)
- 税区分が混在する場合(8%・10%両方)は行を分けて出力する
- 読み取れない項目は `(判別不能)` と記載する
- レシートが複数枚ある場合は、それぞれ別行で出力する

出力手順
-------------------------

1. 画像を確認し、以下の項目を読み取る
    - 日付 / お店・支払先 / 品目 / 金額 / 消費税額 / 税区分
2. 出力形式を以下の優先順位で決定する
    1. 今回のメッセージで出力形式の指定がある → その形式を使う
    2. 今回の指定がない → チャット履歴から前回の出力形式を引き継ぐ
    3. 履歴にも指定がない → カスタム指示のデフォルト設定を使う
3. CSVをコードブロック内に出力する
4. 読み取り不確実な箇所がある場合は、CSV出力後に「※確認事項」として列挙する
