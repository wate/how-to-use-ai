イラスト生成ルール
=========================

ユーザーからイラスト・アイコン・画像生成の依頼があった場合は、
必ずこのファイルのルールに従って処理すること。
対象: SNSアイコン・ブログ挿絵・記事用イラスト・アイキャッチ画像など

イラストの種類と用途判定
-------------------------

素材・依頼内容から種類を判定し、対応するプロンプト構成を使う。

| 種類                 | 判定の手がかり                                         |
| -------------------- | ------------------------------------------------------ |
| SNSアイコン          | 「アイコン」「プロフィール画像」「X」「Instagram」など |
| ブログ・記事用挿絵   | 「記事に使う」「ブログ」「アイキャッチ」など           |
| キャラクター・人物   | 「キャラクター」「人物」「似顔絵」など                 |
| テーマ・概念イラスト | キーワードのみ・テーマのみ(例: 「マグロ」「春」)      |

スタイル判定
-------------------------

ビジュアル設定ファイルに記載がある場合はそれを優先する。
記載がない場合は用途から以下のルールで推定する。

| 用途                 | 推定スタイル                          |
| -------------------- | ------------------------------------- |
| SNSアイコン          | シンプル・フラット・線画              |
| ブログ・記事用挿絵   | フラットデザイン・アイコン風          |
| アイキャッチ画像     | 明るい配色・テキスト入り構成          |
| テーマ・概念イラスト | 水彩風・イラスト風(親しみやすい印象) |

サイズ・縦横比の目安
-------------------------

用途に応じて以下を目安とする。指定がある場合は指定を優先する。

| 用途                        | 推奨サイズ・縦横比          |
| --------------------------- | --------------------------- |
| SNSアイコン(X・Instagram等) | 1:1(正方形)                 |
| ブログ挿絵                  | 16:9 または 4:3             |
| アイキャッチ画像            | 16:9(横長)                  |
| テーマ・概念イラスト        | 指定なし(1:1 または 4:3)    |

プロンプト構成
-------------------------

以下の要素を組み合わせてプロンプトを構成すること。

```
[主題]: 何を描くか
[スタイル]: イラスト風・フラット・水彩風・写真風・線画 など
[色調]: メインカラー・配色の方向性
[構図]: 全体像・アップ・俯瞰 など
[用途]: SNSアイコン・ブログ挿絵 など
[追加条件]: 背景色・テキストの有無 など
```

禁止事項
-------------------------

- 特定の実在する個人を写実的に模倣した画像の生成
- 著作権・肖像権を侵害する可能性がある画像(既存キャラクターの模倣など)
- 暴力・性的表現を含む画像の生成

出力例
-------------------------

### 例1: SNSアイコン(ビジュアル設定ファイルあり)

#### 依頼

```
Xのプロフィールアイコンを作って
```

#### ビジュアル設定ファイルの参照内容

- アカウント: X @example(料理ブロガー)
- テイスト: シンプル・フラット
- イメージカラー: オレンジ系(#FF6B35)

#### 生成プロンプト(AIへの指示)

```
シンプルなフラットデザインのSNSプロフィールアイコン。
料理をテーマにしたイラスト。フォークとスプーンのアイコン、またはお皿のイラスト。
メインカラー: オレンジ(#FF6B35)、背景: 白または薄いオレンジ。
1:1の正方形構図。線は細め、余白を活かしたミニマルなデザイン。
```

※ ビジュアル設定ファイルのアカウント情報・テイスト・カラーを参照して処理しました。

---

### 例2: テーマ・概念イラスト(キーワードのみ)

#### 依頼

```
マグロをテーマにしたイラストを作って
```

#### 生成プロンプト(AIへの指示)

```
マグロをテーマにした水彩風イラスト。
大きなマグロの魚体・刺身・漁船・市場の競りなど、マグロに関連する要素を盛り込む。
和風テイスト、青と赤を基調とした配色。
横長(16:9)構図。親しみやすいイラストスタイル。
```

※ スタイルを「水彩風・和風テイスト」と解釈して処理しました。

出力手順
-------------------------

1. 依頼内容からイラストの種類を判定する
2. ビジュアル設定ファイルを参照し、スタイル・カラーを確認する
3. ファイルに記載がない場合は用途からスタイルを推定する
4. 用途に応じたサイズ・縦横比を確認する(指定がある場合は指定を優先)
5. プロンプト構成の要素を埋めてプロンプトを作成する
6. プロンプトを出力する
7. 推定した内容があれば「※〇〇と解釈して処理しました」と添える
