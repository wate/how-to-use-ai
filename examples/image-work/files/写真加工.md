写真加工ルール
=========================

ユーザーから写真・画像の加工・編集の依頼があった場合は、
必ずこのファイルのルールに従って処理すること。
対象: ブログ・SNS投稿用の写真加工・背景変更・色調補正・不要物除去など

加工の種類と判定
-------------------------

依頼内容から加工の種類を判定し、対応するプロンプト構成を使う。

| 種類         | 判定の手がかり                                     |
| ------------ | -------------------------------------------------- |
| 背景変更     | 「背景を変えて」「背景を白に」「背景を削除」など   |
| 色調補正     | 「明るくして」「暖かい雰囲気に」「モノクロに」など |
| 不要物除去   | 「〇〇を消して」「邪魔なものを消して」など         |
| スタイル変換 | 「イラスト風に」「水彩風に」「ポスター風に」など   |
| トリミング   | 「正方形に切り取って」「顔を中心に」など           |

元素材の保持ルール
-------------------------

加工時は以下のルールに従い、元素材の意図を壊さないこと。

- 人物が写っている場合: 顔・表情・体型を勝手に変えない
- 商品写真の場合: 商品の色・形・テクスチャを正確に保持する
- 背景変更の場合: 主題(人物・商品など)の輪郭を自然に保つ
- 色調補正の場合: 補正の強度は「自然な仕上がり」を基本とし、過剰な加工を避ける

プロンプト構成
-------------------------

以下の要素を組み合わせてプロンプトを構成すること。

```
[元素材の説明]: 写っているもの・状況
[加工の種類]: 背景変更・色調補正・不要物除去 など
[加工の内容]: 具体的にどう変えるか
[保持すべき要素]: 変えてはいけないもの
[仕上がりのイメージ]: 自然な仕上がり・ドラマチック など
```

禁止事項
-------------------------

- 人物の顔・体型・肌の色を無断で変える加工
- 元素材に写っていない人物・物体の追加(依頼がある場合は必ず確認する)
- 著作権・肖像権を侵害する可能性がある加工
- 本人の同意なく特定個人の容姿を変える加工

出力例
-------------------------

### 例1: 背景変更

#### 依頼

```
この写真の背景を白に変えて(商品写真を添付)
```

#### 生成プロンプト(AIへの指示)

```
添付の商品写真の背景を白(#FFFFFF)に変更する。
商品の輪郭を自然に保ち、切り抜きの境界線が不自然にならないようにする。
商品の色・形・テクスチャはそのまま保持すること。
影は薄く自然な形で残してもよい。
```

※ 商品写真のため、商品の色・形を正確に保持するルールを適用しました。

---

### 例2: 色調補正

#### 依頼

```
社員旅行の写真をブログ用に明るく補正して(写真を添付)
```

#### 生成プロンプト(AIへの指示)

```
添付の写真の色調を明るく自然に補正する。
露出を少し上げ、影の部分を持ち上げて全体的に明るい印象にする。
色温度はやや暖かめに調整し、旅行の楽しい雰囲気を引き出す。
人物の肌色・表情は自然な仕上がりを保つこと。過剰な加工は避ける。
```

※ 「ブログ用・明るく」という指定から、自然な仕上がりを基本として処理しました。

出力手順
-------------------------

1. 添付された写真・依頼内容から加工の種類を判定する
2. 元素材の主題(人物・商品・風景など)を確認する
3. 元素材の保持ルールを確認し、変えてはいけない要素を特定する
4. プロンプト構成の要素を埋めてプロンプトを作成する
5. プロンプトを出力する
6. 推定した内容があれば「※〇〇と解釈して処理しました」と添える
