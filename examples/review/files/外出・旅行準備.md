外出・旅行準備ルール
=========================

ユーザーから旅行・引っ越し・お出かけ前の荷物確認依頼があった場合は、必ずこのファイルのルールに従って処理すること。

用途の判定
-------------------------

ユーザーの入力から用途を判定し、対応するセクションのルールを適用する。

| 手がかり                                         | 用途           |
| ------------------------------------------------ | -------------- |
| 「旅行」「出張」「泊まり」「ホテル」             | 旅行・出張     |
| 「引っ越し」「新居」「荷造り」                   | 引っ越し       |
| 「お出かけ」「日帰り」「ピクニック」「キャンプ」 | 日帰り外出     |
| 判断できない                                     | ユーザーに確認 |

必要な情報の確認
-------------------------

処理前に以下の情報が揃っているか確認する。不足している場合は確認してから処理する。

| 情報           | 旅行・出張     | 引っ越し | 日帰り外出     |
| -------------- | -------------- | -------- | -------------- |
| 目的地・行き先 | 必須           | 不要     | あれば望ましい |
| 日数・泊数     | 必須           | 不要     | 不要           |
| 季節・気温     | あれば望ましい | 不要     | あれば望ましい |
| 目的・活動内容 | あれば望ましい | 不要     | 必須           |

旅行・出張
-------------------------

### 見直し観点

| 観点           | 確認内容                                                        |
| -------------- | --------------------------------------------------------------- |
| 必需品の過不足 | 宿泊日数・行き先に対して必要なものが揃っているか                |
| 重複・過剰     | 同じ用途のものを複数持っていないか / 現地調達できるものはないか |
| 荷物の軽量化   | 不要なものや現地で手に入るものを減らせないか                    |
| 忘れ物リスク   | 「ないと困る」ものが漏れていないか                              |

### 基本チェックリスト

行き先・日数に応じて以下を確認する。

| カテゴリ   | 品目                                                        |
| ---------- | ----------------------------------------------------------- |
| 衣類       | 着替え(日数分) / 下着・靴下 / 羽織もの / 寝間着(必要な場合) |
| 身だしなみ | 洗面用具 / スキンケア / 歯ブラシ / 化粧品(必要な場合)       |
| デジタル   | スマホ / 充電器 / モバイルバッテリー / イヤホン             |
| 書類・決済 | 財布 / クレカ / 身分証 / 保険証 / チケット・予約確認        |
| 健康・衛生 | 常備薬 / 酔い止め(必要な場合) / マスク                      |
| その他     | 傘・雨具 / サングラス(行き先による) / 変換アダプタ(海外)    |

### 出力フォーマット

#### 持ち物リスト確認

| カテゴリ | 品目 | 状態               | コメント |
| -------- | ---- | ------------------ | -------- |
| 〇〇     | 〇〇 | OK / 不足 / 要確認 | 〇〇     |

#### 追加推奨

ユーザーのリストに含まれていないが、行き先・日数・季節から推奨される品目を提案する。

#### 削減提案

現地調達可能・不要と判断されるものがある場合に提案する。

### 出力例

素材: 3泊4日の北海道旅行(10月)/ 荷物: 着替え3セット・充電器・財布・常備薬

出力:

```
持ち物リスト確認(3泊4日・北海道・10月)

| カテゴリ   | 品目            | 状態   | コメント                         |
| ---------- | --------------- | ------ | -------------------------------- |
| 衣類       | 着替え(3セット) | 要確認 | 4日分あると安心。予備1セット推奨 |
| デジタル   | 充電器          | OK     |                                  |
| 書類・決済 | 財布            | OK     | クレカ・身分証の確認を           |
| 健康       | 常備薬          | OK     |                                  |

追加推奨:
- 厚手の羽織もの・コート: 10月の北海道は最低気温が10度を下回る日も
- モバイルバッテリー: 観光中の充電切れ対策に
- 雨具: 天気が変わりやすい季節
- 歯ブラシ・洗面用具: リストに記載がなかったため確認を
- 保険証(またはコピー): 念のため

※ 最終的な持ち物の判断はご自身でお願いします。
   天気予報も直前に確認してください。
```

引っ越し
-------------------------

### 見直し観点

| 観点             | 確認内容                                                   |
| ---------------- | ---------------------------------------------------------- |
| 荷造りの優先順位 | 直前まで使うもの・早めに梱包できるものの仕分け             |
| 手続き確認       | 住所変更・各種解約・転入届など必要な手続きが漏れていないか |
| 処分・整理       | 引っ越し前に手放せるものはないか                           |

### 出力フォーマット

#### 荷造り状況確認

| カテゴリ | 状態             | コメント |
| -------- | ---------------- | -------- |
| 〇〇     | 済 / 未 / 要確認 | 〇〇     |

#### 手続きチェックリスト(主なもの)

| 手続き                 | 目安タイミング     |
| ---------------------- | ------------------ |
| 転出届                 | 引っ越し前         |
| 転入届                 | 引っ越し後14日以内 |
| 運転免許証の住所変更   | 引っ越し後早めに   |
| 郵便物の転送手続き     | 引っ越し前         |
| 各種サービスの住所変更 | 引っ越し前後       |
| 電気・ガス・水道       | 引っ越し前後       |

### 注記ルール

出力の末尾に必ず以下を添える。

```
※ 手続きの詳細・期限は各機関にご確認ください。
```

日帰り外出
-------------------------

活動内容によって必要な持ち物が大きく変わるため、活動内容を確認してから処理する。
活動内容が明示されていない場合は「どんなお出かけですか?(例: ピクニック・キャンプ・買い物)」と確認すること。

### 活動別チェックリスト

活動内容に応じて以下のカテゴリから必要なものを選んで確認する。

| カテゴリ       | 品目                                                       | 主な用途           |
| -------------- | ---------------------------------------------------------- | ------------------ |
| 共通(全活動)   | 財布・スマホ・交通系ICカード・充電器・飲み物               | 全活動共通         |
| 食事・飲み物   | 水筒・お弁当・おやつ・保冷バッグ                           | ピクニック・遠足   |
| 屋外・自然     | 日焼け止め・虫除け・帽子・タオル・着替え・雨具             | ピクニック・ハイキング |
| キャンプ特有   | テント・寝袋・調理器具・ランタン・着火道具・ゴミ袋         | キャンプ           |
| 子ども連れ     | おむつ・着替え・おもちゃ・ウェットティッシュ・常備薬       | 子ども連れ全般     |
| 健康・安全     | 救急セット・常備薬・モバイルバッテリー                     | 長時間外出全般     |

### 出力フォーマット

#### 持ち物確認

| 持ち物 | 状態               | コメント |
| ------ | ------------------ | -------- |
| 〇〇   | OK / 不足 / 要確認 | 〇〇     |

#### 追加推奨

活動内容・行き先・季節から、リストに含まれていない推奨品目を提案する。

### 出力例

素材: 家族でピクニック(公園・5月)/ 荷物: お弁当・レジャーシート・スマホ・財布

出力:

```
持ち物確認(ピクニック・公園・5月)

| 持ち物         | 状態   | コメント               |
| -------------- | ------ | ---------------------- |
| お弁当         | OK     |                        |
| レジャーシート | OK     |                        |
| スマホ         | OK     |                        |
| 財布           | OK     |                        |

追加推奨:
- 飲み物・水筒: 5月は気温が上がる日も。多めに用意を
- 日焼け止め: 屋外長時間はあると安心
- 虫除けスプレー: 公園は蚊が出始める季節
- ゴミ袋: 持ち帰り用に1〜2枚
- ウェットティッシュ: 食事前後や汚れたときに

※ 最終的な持ち物の判断はご自身でお願いします。
```

共通の注記ルール
-------------------------

出力の末尾に必ず以下を添える(引っ越しを除く)。

```
※ 最終的な持ち物の判断はご自身でお願いします。
```

出力手順
-------------------------

1. 入力から用途(旅行・出張 / 引っ越し / 日帰り外出)を判定する
2. 必要な情報(目的地・日数・活動内容など)が揃っているか確認する
3. 対応するセクションの観点・チェックリストを適用する
4. フォーマットに従って出力する
5. 注記を添える
