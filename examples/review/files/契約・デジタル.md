契約・デジタル見直しルール
=========================

ユーザーからサブスク・契約・スマホアプリの見直し依頼があった場合は、必ずこのファイルのルールに従って処理すること。

用途の判定
-------------------------

ユーザーの入力から用途を判定し、対応するセクションのルールを適用する。

| 手がかり                                       | 用途           |
| ---------------------------------------------- | -------------- |
| 「サブスク」「契約」「明細」「固定費」「保険」 | サブスク・契約 |
| 「アプリ」「ホーム画面」「スクショ」「通知」   | スマホアプリ   |
| 判断できない                                   | ユーザーに確認 |

サブスク・契約見直し
-------------------------

### 見直し観点

以下の観点で整理する。

| 観点         | 確認内容                                                 |
| ------------ | -------------------------------------------------------- |
| 利用頻度     | 月に何回使っているか。ほぼ使っていないものはないか       |
| コスト妥当性 | 月額・年額に対して得られている価値は見合っているか       |
| 重複         | 同じ用途をカバーしているサービスが複数ないか             |
| 無料代替     | 無料プランや他の手段で代替できるものはないか             |
| 利用実態     | 登録したことを忘れていたり、惰性で続けているものはないか |

### 判断基準

| 判断     | 基準                                         |
| -------- | -------------------------------------------- |
| 継続     | 月1回以上利用 かつ コストに見合う価値がある  |
| 要検討   | 利用頻度が低い / 似たサービスと重複している  |
| 解約候補 | ほぼ未使用 / 無料代替あり / 存在を忘れていた |

### 不足情報の扱い

以下の情報が不足している場合は確認してから処理する。

- 月額・年額が不明な場合: 「金額がわかるものがあれば教えてください」と確認する
- 利用頻度が不明な場合: 推定せず「最近使いましたか?」と確認する
- サービス名だけで用途が不明な場合: 用途を推定して処理し、「※〇〇は〇〇として整理しました」と添える

### 出力フォーマット

#### 契約一覧(表形式)

| サービス名 | 月額(税込) | 用途 | 利用頻度 | 判断                     |
| ---------- | ---------- | ---- | -------- | ------------------------ |
| 〇〇       | 〇〇円     | 〇〇 | 月〇回   | 継続 / 要検討 / 解約候補 |

#### 見直し提案

- 解約候補: (サービス名) — 理由
- 要検討: (サービス名) — 理由・代替案

#### 合計・削減試算

- 現在の月額合計: 〇〇円
- 解約候補の合計: 〇〇円/月
- 削減後の月額合計(目安): 〇〇円

### 注記ルール

出力の末尾に必ず以下を添える。

```
※ 解約・プラン変更の最終判断はご自身でお願いします。
   年払い契約は解約タイミングにご注意ください。
```

### 出力例

素材: クレカ明細のスクショ(Netflix 1,490円・Spotify 980円・使っていないジム 8,000円・Adobe 3,280円)

出力:

```
契約一覧

| サービス名 | 月額(税込) | 用途          | 利用頻度   | 判断     |
| ---------- | ---------- | ------------- | ---------- | -------- |
| Netflix    | 1,490円    | 動画配信      | 不明       | 要確認   |
| Spotify    | 980円      | 音楽配信      | 不明       | 要確認   |
| ジム       | 8,000円    | フィットネス  | ほぼ未使用 | 解約候補 |
| Adobe CC   | 3,280円    | デザイン・PDF | 不明       | 要確認   |

月額合計: 13,750円

見直し提案:
- 解約候補: ジム(8,000円) — ほぼ未使用とのこと。解約または休会で削減できます
- Netflix・Spotify: 利用頻度を教えていただければ、さらに整理できます

削減試算: ジム解約のみで月 -8,000円(年 -96,000円)

※ 解約・プラン変更の最終判断はご自身でお願いします。
   年払い契約は解約タイミングにご注意ください。
```

スマホアプリ整理
-------------------------

### 見直し観点

| 観点         | 確認内容                                             |
| ------------ | ---------------------------------------------------- |
| 利用頻度     | 最後に開いたのはいつか。ほぼ使っていないものはないか |
| 課金状況     | 無料のつもりで課金が発生しているものはないか         |
| 通知過多     | 通知が多く邪魔になっているアプリはないか             |
| 重複         | 同じ用途のアプリが複数入っていないか                 |
| セキュリティ | 長期間更新されていない・提供元不明のアプリはないか   |

### 判断基準

| 判断     | 基準                                              |
| -------- | ------------------------------------------------- |
| 継続     | 週1回以上使用 または 必要なときだけ使う系のアプリ |
| 要検討   | 月1回未満 / 似た用途のアプリと重複                |
| 削除候補 | ほぼ未使用 / 課金発生しているが使っていない       |
| 通知整理 | 使うが通知が不要・過多なもの                      |

### 出力フォーマット

#### アプリ整理結果

| アプリ名 | 用途 | 課金               | 判断                                |
| -------- | ---- | ------------------ | ----------------------------------- |
| 〇〇     | 〇〇 | あり / なし / 不明 | 継続 / 要検討 / 削除候補 / 通知整理 |

#### 提案

- 削除候補: (アプリ名) — 理由
- 通知整理: (アプリ名) — 通知をオフにすることを推奨
- 要検討: (アプリ名) — 理由

### 注記ルール

出力の末尾に必ず以下を添える。

```
※ アプリ削除・課金解除の最終判断はご自身でお願いします。
   課金アプリは削除前にサブスク解除をご確認ください。
```

出力手順
-------------------------

1. 入力から用途(サブスク・契約 / スマホアプリ)を判定する
2. 不足情報を確認する(金額・利用頻度など)
3. 各観点・判断基準に従って整理する
4. フォーマットに従って出力する
5. 注記を添える
6. 推定した内容がある場合は「※〇〇と判断しました」と明示する
