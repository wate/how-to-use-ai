モノ整理ルール
=========================

ユーザーからクローゼット・バッグ・部屋・防災用品などの持ち物整理依頼があった場合は、必ずこのファイルのルールに従って処理すること。

用途の判定
-------------------------

ユーザーの入力から用途を判定し、対応するセクションのルールを適用する。

| 手がかり                                     | 用途           |
| -------------------------------------------- | -------------- |
| 「クローゼット」「服」「着ていない」「季節」 | クローゼット   |
| 「バッグ」「カバン」「持ち歩き」             | バッグの中身   |
| 「防災」「備蓄」「非常用」「期限」           | 防災用品       |
| 「部屋」「デスク」「机」「収納」             | 部屋・デスク   |
| 判断できない                                 | ユーザーに確認 |

共通の判断基準
-------------------------

持ち物全般に適用する判断基準。

| 判断       | 基準                                                          |
| ---------- | ------------------------------------------------------------- |
| 残す       | 直近1年以内に使った / 今後確実に使う予定がある / 代替がない   |
| 要検討     | 1年以上使っていないが思い入れがある / 使う機会が来るか不明    |
| 手放す候補 | 1年以上使っていない / 同じ用途のものが複数ある / 劣化している |

「要検討」に対しては処分を急かさず、「次の季節まで様子を見る」などの提案にとどめる。

クローゼット整理
-------------------------

### 見直し観点

| 観点         | 確認内容                                  |
| ------------ | ----------------------------------------- |
| 着用頻度     | 直近1年間で着たか                         |
| 用途の重複   | 同じ場面で着られる服が複数ないか          |
| サイズ・状態 | 今も着られるサイズか / くたびれていないか |
| 季節適合     | 今の季節・ライフスタイルに合っているか    |

### 出力フォーマット

#### 整理結果

| アイテム | 状態 | 判断                       | コメント |
| -------- | ---- | -------------------------- | -------- |
| 〇〇     | 〇〇 | 残す / 要検討 / 手放す候補 | 〇〇     |

#### 提案

- 手放す候補: (アイテム) — 理由
- 要検討: (アイテム) — 提案(例: 次の季節まで様子を見る)

### 出力例

素材: クローゼットの写真(黒いスーツ・3年前のコート・同じような白シャツ3枚・ほぼ着ていないワンピース)

出力:

```
整理結果

| アイテム      | 判断       | コメント                         |
| ------------- | ---------- | -------------------------------- |
| 黒いスーツ    | 残す       | フォーマルな場面で必要           |
| 3年前のコート | 要検討     | 着用感・状態次第。試着してみて   |
| 白シャツ(3枚) | 要検討     | 1〜2枚に絞れるか確認を           |
| ワンピース    | 手放す候補 | ほぼ未着用。フリマ・寄付も選択肢 |

提案:
- 手放す候補: ワンピース(ほぼ未着用)
- 要検討: コート(状態確認)・白シャツ(枚数を絞る)

※ 最終的に残す・手放すかはご自身でご判断ください。
```

バッグの中身整理
-------------------------

### 見直し観点

| 観点   | 確認内容                                               |
| ------ | ------------------------------------------------------ |
| 必要性 | 毎日・週に何回使うか                                   |
| 重複   | 同じ用途のものが複数入っていないか                     |
| 緊急性 | なくても問題ないが入れっぱなしになっているものはないか |
| 重さ   | 不要なものを減らして軽くできないか                     |

### 出力フォーマット

#### 整理結果

| 持ち物 | 利用頻度                | 判断                 | コメント |
| ------ | ----------------------- | -------------------- | -------- |
| 〇〇   | 毎日 / 週1〜 / ほぼなし | 必要 / 要検討 / 不要 | 〇〇     |

#### 提案

- 取り出してよいもの: (アイテム) — 理由
- 追加を検討: (アイテム) — 理由(素材に不足が見られる場合のみ)

防災用品見直し
-------------------------

### 見直し観点

| 観点         | 確認内容                                     |
| ------------ | -------------------------------------------- |
| 品目の過不足 | 基本的な防災用品が揃っているか               |
| 期限         | 食料・飲料・医薬品の消費期限は切れていないか |
| 状態         | 電池・ライト・ラジオは動作するか             |
| 人数対応     | 家族の人数分揃っているか                     |

### 基本チェックリスト

以下の品目を基準に不足・期限切れを確認する。

| カテゴリ   | 品目                                       |
| ---------- | ------------------------------------------ |
| 水・食料   | 飲料水(1人1日3L目安)・非常食(3日分以上)    |
| 救急・衛生 | 救急セット・常備薬・マスク・消毒液         |
| 情報・通信 | ラジオ・モバイルバッテリー・充電ケーブル   |
| 照明       | 懐中電灯・ランタン・予備電池               |
| 避難用品   | 非常持ち出し袋・雨具・軍手・笛             |
| 書類       | 保険証のコピー・通帳のコピー・連絡先リスト |

### 出力フォーマット

#### 確認結果

| カテゴリ | 品目 | 状態                   | 判断                     |
| -------- | ---- | ---------------------- | ------------------------ |
| 〇〇     | 〇〇 | あり / なし / 期限切れ | OK / 補充が必要 / 要確認 |

#### 提案

- 補充が必要: (品目) — 理由
- 要確認: (品目) — 確認内容

### 注記ルール

出力の末尾に必ず以下を添える。

```
※ 防災用品は半年〜1年に1回の点検を推奨します。
   家族の人数・持病・ペットの有無に合わせて内容をカスタマイズしてください。
```

部屋・デスク整理
-------------------------

### 見直し観点

| 観点       | 確認内容                                     |
| ---------- | -------------------------------------------- |
| 使用頻度   | よく使うものが取り出しやすい位置にあるか     |
| 不要物     | 使っていないものが場所を占有していないか     |
| 収納の適正 | モノの量が収納スペースに対して多すぎないか   |
| 作業環境   | 作業・生活のしやすさを妨げている配置はないか |

### 出力フォーマット

#### 整理提案

- 取り除く候補: (モノ) — 理由
- 配置の見直し: (場所・モノ) — 提案
- 収納の改善: (場所) — 提案

共通の注記ルール
-------------------------

出力の末尾に必ず以下を添える。

```
※ 処分・手放すかどうかの最終判断はご自身でお願いします。
```

出力手順
-------------------------

1. 入力から用途(クローゼット / バッグ / 防災 / 部屋・デスク)を判定する
2. 対応するセクションの観点・判断基準を適用する
3. フォーマットに従って出力する
4. 注記を添える
5. 写真・画像からの読み取りの場合、確認できた範囲で処理し「※写真から確認できた範囲での整理です」と添える
