見直し・整理
=========================

固定費・契約・持ち物・食材など、定期的に見直したいことを
AIと一緒に棚卸し・整理する活用事例です。

写真・スクショ・メモ・PDFをそのまま投げるだけで、登録したファイルの観点に従って
AIが用途を自動判定し、整理・見直し提案を出力します。

-------------------------

| ---------------- | -------------- | ----------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.5 Instant   |

セットアップ(初回のみ)
-------------------------

> プロジェクト の概要や共通の考え方は [活用事例 README](../README.md) を参照してください。

### 基本的な流れ

1. [プロジェクトを作成](../README.md#プロジェクト機能とは)する(例: 「見直し・整理」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 登録対象ファイルを各プラットフォームの指定の項目に登録する
    - ChatGPT: 「情報源」項目
    - Claude: PC版は「ファイル」、スマートフォン版は「プロジェクトナレッジ」

### 登録対象ファイル

ここでの登録対象ファイルはfiles/以下です。

- [files/契約・デジタル.md](files/契約・デジタル.md)
- [files/モノ.md](files/モノ.md)
- [files/食材.md](files/食材.md)
- [files/外出・旅行準備.md](files/外出・旅行準備.md)

プロフィールを作成した場合は、必要に応じて追加で登録してください。
個人の記録か家庭共有かなどの文脈判断が正確になります。

- 推奨: プロジェクトにファイルとして登録する(プロジェクト単位で文脈を変えたい場合)
- フォールバック: アカウントのプロファイル設定に登録する(全チャットで自動適用、プロジェクトにファイルがない場合に使用される)

作り方は [プロフィール生成プロンプト](../../prompts/プロフィール生成.md) を参照してください。

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# サブスクを見直す場合
クレカ明細のスクショです。サブスクを整理してください。
(スクショを添付)

# 冷蔵庫を棚卸しする場合
冷蔵庫の中です。消費優先度と買い物で足りないものを教えてください。
(写真を添付)

# 旅行前の荷物を確認する場合
3泊4日で沖縄に行きます。荷物候補を確認してください。
- 着替え3セット
- 日焼け止め
- 充電器
```

その他の活用事例は[活用事例](#活用事例)を参照。

チャット管理
-------------------------

- 用途ごとにチャットを分けて管理する
    - 例: 「サブスク整理 2026年5月」「冷蔵庫 2026年5月」
- チャットが長くなったら新チャットに切り替える
- 節目ごと(月末・四半期・季節の変わり目)にチャットを新規作成して棚卸しする

活用事例
-------------------------

- サブスク整理: クレカ明細のスクショを投げて使っていないサービスを洗い出す
- 冷蔵庫の棚卸し: 写真を撮って消費優先度と買い物リストを整理する
- 旅行の持ち物確認: 荷物候補を投げて不足や過不足を確認する
- 持ち物の断捨離: 部屋や収納の写真から残す・手放すを判断する

### 事業者向け活用例

- 業務ツール・サブスクの見直し: 事業で契約しているサービスの利用状況を棚卸しして無駄なコストを削減する
- 備品・在庫の確認: 事務用品や消耗品の写真を撮って補充リストを整理する
- 定期契約の条件確認: 更新時期が近い契約の内容を整理して見直しポイントを把握する
