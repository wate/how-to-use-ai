相談・考え整理 設計資料
=========================

目的
-------------------------

専門知識がなくて判断できない場面で、一般ユーザーが気軽に相談・整理できる
AIアシスタントの構成を定義する。

対象ユーザー
-------------------------

- 非エンジニアの一般ユーザー
- 個人・家庭での意思決定に迷っている人
- 個人事業主・小規模事業者
- キャリア・人生の選択に悩んでいる人

設計の核心
-------------------------

- ユーザーがやること: 状況・文書・悩みを投げるだけ
- AIがやること: ユースケースを判定し、適切なフレームワークを使って整理・提示する
- ユーザーが毎回入力する必要があるもの: 原則として状況説明または文書のみ

フレームワークの名前をユーザーが知っている必要はない。
AIが文脈から判断して自動的に使うフレームワークを選ぶ設計とする。

設計方針
-------------------------

### 用途別ファイル分割

相談内容によって「良い出力」の基準が異なるため、用途単位でファイルを分割する。
AIが相談内容からどのファイルを参照すべきかを自動判定する。

### 専門家誘導ルールを共通で持つ

健康・法律・メンタルヘルスなど専門資格が必要な領域は、
一般的な情報・整理として対応しつつ、出力の末尾に専門家への確認を促す一文を添える。
完全に突っぱねる設計はしない。

### カスタム指示はAIへの動作命令として機能させる

- ユースケースの自動判定ルール
- 不足情報があるときの推定・確認のルール
- 登録済みファイルの参照ルール
- 専門家誘導の判定ルール

ファイル構成
-------------------------

```
カスタム指示.md                  <- カスタム指示欄に貼り付ける。動作命令を記述する
README.md                        <- ユースケース棚卸し・セットアップ手順・日常利用例
files/文書確認.md                <- 契約約款・見積書・規約などのリスク確認ルール
files/意思決定.md                <- プロコン、優先度マトリクス、ディシジョンツリー、SWOT、3C
files/アイデア出し.md            <- ブレインストーミング、マンダラート、SCAMPER
files/問題整理・相談.md          <- なぜなぜ分析、KJ法、SMARTゴール、GROWモデル、Will-Can-Must
```

各ファイルに必須の記載事項
-------------------------

1. このファイルを参照するタイミング(AIへの指示)
2. 含むフレームワークの一覧と使い分け基準
3. 各フレームワークの出力フォーマット定義
4. 出力手順(AIが従うステップ)
5. 専門家誘導が必要な場合の判定基準(文書確認・問題整理・相談に記載)

適用範囲と制約
-------------------------

- 対象: 日常・仕事問わず一般ユーザーが直面する相談・意思決定・問題整理
- 対象外: 高度な専門判断(医療診断・法的判断・税務申告など)
- 専門家が必要な領域は情報提供にとどめ、最終判断は専門家に委ねる
- AIの回答は参考情報であり、最終判断はユーザー自身が行う

設計判断ログ
-------------------------

| 判断                                     | 理由                                                                  |
| ---------------------------------------- | --------------------------------------------------------------------- |
| フレームワーク名をユーザーに意識させない | 非エンジニア向けのため、AIが自動選択する設計とする                    |
| 専門家誘導を完全拒否にしない             | 突っぱねると使い物にならないため、情報提供+誘導の形とする             |
| ビジネス戦略系の一部(SWOT・3C)を残す     | 個人事業主・小規模事業者の利用場面をカバーするため                    |
| 高度な組織マネジメント系は除外           | 対象ユーザーの利用場面に合わないため                                  |
| Will-Can-Mustを追加                      | キャリア・人生相談との相性がよく、SMARTゴールと組み合わせて使えるため |
