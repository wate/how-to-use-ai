文書確認
=========================

このファイルを参照するタイミング
-------------------------

以下のいずれかに該当する場合にこのファイルを参照すること。

- 契約書・約款・利用規約・同意書などの文書が添付または引用されている
- 「確認してほしい」「問題ないか見てほしい」「不利な条項がないか調べてほしい」などの依頼
- 見積書・請求書・発注書などのビジネス文書の内容確認

処理の手順
-------------------------

1. 文書の種類を判定する(契約書 / 利用規約 / 見積書 / その他)
2. 文書全体を読み、以下の観点でリスクを洗い出す
3. リスクの重要度を3段階(高 / 中 / 低)で分類して出力する
4. 不明瞭な条項・見落としやすい記述を明示する
5. 専門家への確認が必要な箇所を明示する

確認観点
-------------------------

### 契約書・約款・利用規約

- 自動更新・自動継続の条項(解約手続きの期限・方法)
- 違約金・損害賠償の範囲と上限
- 個人情報・データの取り扱いと第三者提供の条件
- サービス変更・終了時のユーザーへの影響
- 準拠法・裁判管轄(遠方の裁判所が指定されていないか)
- 免責事項の範囲(事業者側が責任を負わない範囲が広すぎないか)
- 禁止事項と違反時のペナルティ
- 小さく記載されている条項・脚注・別紙への参照

### 見積書・請求書・発注書

- 金額の内訳(税込 / 税抜の明示)
- 支払い条件・期日・遅延損害金
- 納期・納品条件
- キャンセル・変更時の費用負担
- 追加費用が発生する条件

出力フォーマット
-------------------------

```
## 文書の種類
(判定した文書の種類)

## 確認結果サマリー
(全体的な印象と注意度を1〜2文で)

## リスク一覧

### 重要度: 高
- (条項名または箇所): (内容と問題点)

### 重要度: 中
- (条項名または箇所): (内容と注意点)

### 重要度: 低
- (条項名または箇所): (内容と補足)

## 不明瞭・見落としやすい箇所
- (箇所): (なぜ注意が必要か)

## 専門家への確認推奨箇所
- (箇所): (確認を勧める理由)
```

専門家誘導の判定基準
-------------------------

以下に該当する場合は出力末尾に専門家誘導を添えること。

- 金額が大きい契約(目安: 10万円以上)
- 長期間の契約(1年以上の継続義務)
- 個人情報・知的財産権に関わる条項が含まれる
- 違約金・損害賠償の条項が含まれる
- 雇用・業務委託に関わる契約
- ユーザーが「署名・捺印する前に確認したい」と述べている場合

注意事項
-------------------------

- このファイルの分析はあくまで一般的な観点からのリスク整理であり、法的判断ではない
- 最終的な契約判断は必ずユーザー自身が行うこと
- 重要な契約は弁護士・司法書士への確認を推奨する
