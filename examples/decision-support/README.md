相談・考え整理
=========================

契約書のリスク確認・選択肢の比較・アイデア出し・キャリア相談など、
「知識がなくて判断できない」場面で気軽に相談できる活用事例です。

状況や文書を投げるだけで、AIが内容から用途を判断し、
適切な方法で整理・提示します。

-------------------------

| プラットフォーム | 想定プラン     | モデル            |
| ---------------- | -------------- | ----------------- |

セットアップ(初回のみ)
-------------------------

> プロジェクト の概要や共通の考え方は [活用事例 README](../README.md) を参照してください。

### 基本的な流れ

1. [プロジェクトを作成](../README.md#プロジェクト機能とは)する(例: 「相談・考え整理」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 登録対象ファイルを各プラットフォームの指定の項目に登録する
    - ChatGPT: 「情報源」項目
    - Claude: PC版は「ファイル」、スマートフォン版は「プロジェクトナレッジ」

### 登録対象ファイル

ここでの登録対象ファイルはfiles/以下です。

- [files/文書確認.md](files/文書確認.md)
- [files/意思決定.md](files/意思決定.md)
- [files/アイデア出し.md](files/アイデア出し.md)
- [files/問題整理・相談.md](files/問題整理・相談.md)

プロフィールを作成した場合は、必要に応じて追加で登録してください。
プライベートか仕事かの文脈判断が正確になります。

- 推奨: プロジェクトにファイルとして登録する(プロジェクト単位で文脈を変えたい場合)
- フォールバック: アカウントのプロファイル設定に登録する(全チャットで自動適用、プロジェクトにファイルがない場合に使用される)

作り方は [プロフィール生成プロンプト](../../prompts/プロフィール生成.md) を参照してください。

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# 契約書を確認してもらう場合
この契約書を確認して(ファイルを添付または内容を貼り付け)

# 選択肢を比較する場合
転職するかどうか迷っています
- 現職: 安定しているが給料が低く成長を感じない
- 転職先候補: 給料は上がるが不安定・残業が多い

# 優先順位をつける場合
やりたいことが多すぎて整理できません(リストを貼り付け)

# アイデアを出す場合
副業でできそうなことをアイデア出ししてほしい
自分のスキル: Webデザイン・写真撮影・英語

# 問題の原因を掘り下げる場合
なぜか仕事のやる気が出ない状態が続いています

# キャリアの相談をする場合
30代でエンジニアからマネージャーへの転向を考えています
迷っていることを整理してほしい
```

その他の活用事例は[活用事例](#活用事例)を参照。

専門家への相談について
-------------------------

AIの回答はあくまで整理・参考情報です。
以下の領域に関わる内容は、最終的な判断を専門家に委ねることを推奨します。

| 領域           | 相談先                 |
| -------------- | ---------------------- |
| 健康・医療     | 医師・薬剤師           |
| 法律・契約     | 弁護士・司法書士       |
| 税務・会計     | 税理士・公認会計士     |
| メンタルヘルス | 心療内科・カウンセラー |
| 労務・雇用     | 社会保険労務士         |

チャット管理
-------------------------

- 相談内容ごとにチャットを分けて管理する
    - 例: 「転職の相談 2026年5月」「新規事業アイデア 2026年5月」
- チャットが長くなったら新チャットに切り替える
- 前回の続きから始めたい場合は、最初に「前回の続きで〜」と状況を簡単に補足する

活用事例
-------------------------

- 契約書確認: 難しい条文のリスクや注意点を整理してもらう
- 選択肢の比較: 引っ越し先・転職先・購入品など複数の選択肢を比較して判断を整理する
- アイデア出し: 副業・趣味・企画など、発想を広げたい場面で使う
- キャリア相談: 転職・独立・方向性の迷いを整理する
- 問題の掘り下げ: なんとなくモヤモヤしていることの原因を一緒に考える

### 事業者向け活用例

- 見積書・契約書の確認: 取引先から届いた書類の気になる条件やリスクを整理する
- 取引先との条件比較: 複数の業者・サービスの条件を並べて比較・判断の整理をする
- 新サービスのアイデア出し: 自分のスキルや強みをもとに新しい収益源のアイデアを広げる
- 業務上の悩み相談: 顧客対応・価格設定・スケジュール管理など、日常の判断に迷ったときに相談する
