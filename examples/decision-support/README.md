相談・考え整理
=========================

契約書のリスク確認・選択肢の比較・アイデア出し・キャリア相談など、
「知識がなくて判断できない」場面で気軽に相談できる活用事例です。

状況や文書を投げるだけで、AIが内容から用途を判断し、
適切な方法で整理・提示します。

対象プラットフォーム
-------------------------

| プラットフォーム | 想定プラン     | モデル          |
| ---------------- | -------------- | --------------- |
| ChatGPT          | 無料プラン以上 | GPT-5.5 Instant |
| Gemini           | 無料プラン以上 | Gemini 3 Flash  |

どのファイルを登録するか
-------------------------

ChatGPT 無料プランはプロジェクトへの登録ファイルが **5つまで** です。
全ファイルを登録する必要はありません。以下の棚卸し表を参考に、
自分がよく使う用途のファイルだけ選んでください。

### ユースケース棚卸し表

| 自分の用途                                         | 登録するファイル       |
| -------------------------------------------------- | ---------------------- |
| 契約書・利用規約・見積書を確認したい               | `文書確認.md`          |
| 複数の選択肢を比較して決めたい                     | `意思決定.md`          |
| やることが多すぎて優先順位をつけたい               | `意思決定.md`          |
| 個人事業・小規模事業の状況や方向性を整理したい     | `意思決定.md`          |
| アイデアをたくさん出したい・発想を広げたい         | `アイデア出し.md`      |
| 既存のやり方やサービスを改善・発展させたい         | `アイデア出し.md`      |
| 何が問題かわからない・モヤモヤを整理したい         | `問題整理・相談.md`    |
| キャリア・転職・将来の方向性を相談したい           | `問題整理・相談.md`    |
| 目標を具体化して行動計画を立てたい                 | `問題整理・相談.md`    |

#### 例: 個人事業主(契約確認＋事業方針の検討＋アイデア出し)の場合

```
登録するファイル:
- 文書確認.md       <- 取引先との契約書・見積書の確認
- 意思決定.md       <- 事業の方向性・優先順位の整理
- アイデア出し.md   <- 新サービス・改善アイデアの発散

計3ファイル -> 無料プランでも予備枠が2つ残る
```

セットアップ(初回のみ)
-------------------------

> プロジェクト / Gems の概要や共通の考え方は [活用事例 README](../README.md) を参照してください。

### ChatGPT(プロジェクト機能)

1. プロジェクトを作成する(例: 「相談・考え整理」)
2. `カスタム指示.md` の内容をプロジェクトのカスタム指示欄に貼り付ける
3. 棚卸し表で選んだファイルを情報源として登録する

### Gemini(Gems 機能)

1. Gem を作成する(例: 「相談・考え整理」)
2. `カスタム指示.md` の内容を指示欄に貼り付ける
3. 棚卸し表で選んだファイルをナレッジファイルとして登録する

日常利用
-------------------------

セットアップ後は、プロジェクト／Gem を選択してチャットを開始するだけです。

```
# 契約書を確認してもらう場合
この契約書を確認して(ファイルを添付または内容を貼り付け)

# 選択肢を比較する場合
転職するかどうか迷っています
- 現職: 安定しているが給料が低く成長を感じない
- 転職先候補: 給料は上がるが不安定・残業が多い

# 優先順位をつける場合
やりたいことが多すぎて整理できません(リストを貼り付け)

# アイデアを出す場合
副業でできそうなことをアイデア出ししてほしい
自分のスキル: Webデザイン・写真撮影・英語

# 問題の原因を掘り下げる場合
なぜか仕事のやる気が出ない状態が続いています

# キャリアの相談をする場合
30代でエンジニアからマネージャーへの転向を考えています
迷っていることを整理してほしい
```

専門家への相談について
-------------------------

AIの回答はあくまで整理・参考情報です。
以下の領域に関わる内容は、最終的な判断を専門家に委ねることを推奨します。

| 領域           | 相談先                   |
| -------------- | ------------------------ |
| 健康・医療     | 医師・薬剤師             |
| 法律・契約     | 弁護士・司法書士         |
| 税務・会計     | 税理士・公認会計士       |
| メンタルヘルス | 心療内科・カウンセラー   |
| 労務・雇用     | 社会保険労務士           |

チャット管理
-------------------------

- 相談内容ごとにチャットを分けて管理する
    - 例: 「転職の相談 2026年5月」「新規事業アイデア 2026年5月」
- チャットが長くなったら新チャットに切り替える
- 前回の続きから始めたい場合は、最初に「前回の続きで〜」と状況を簡単に補足する
